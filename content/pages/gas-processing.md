## Gas Processing Plant Manufacturing Process

### Gas Inlet and Reception System

1. Survey site geology for stable foundation requirements
2. Conduct environmental impact assessment for facility placement
3. Obtain regulatory permits from state and federal agencies
4. Clear and level 50-acre site for processing facility
5. Install perimeter fencing and security systems
6. Excavate foundations for main process units
7. Pour reinforced concrete pads for equipment mounting
8. Install underground drainage and storm water management
9. Construct inlet manifold receiving area
10. Install high-pressure inlet separators (3-phase)
11. Mount inlet gas scrubbers for liquid removal
12. Install inlet gas filters for particulate removal
13. Connect pipeline tie-ins from gas gathering system
14. Install inlet pressure control valves and safety systems
15. Mount inlet flow measurement equipment (ultrasonic meters)
16. Install inlet gas sampling systems for quality analysis
17. Connect inlet pig receiver for pipeline cleaning tools
18. Install inlet emergency shutdown valves
19. Mount inlet gas heaters to prevent hydrate formation
20. Connect inlet condensate collection system

### Acid Gas Removal Unit

21. Construct amine treating unit foundation
22. Install amine contactor tower (150 ft tall)
23. Mount structured packing inside contactor column
24. Install amine regenerator tower
25. Connect amine circulation pumps (centrifugal design)
26. Install lean/rich amine heat exchanger
27. Mount amine reboiler with steam heating
28. Install amine cooler with air or water cooling
29. Connect amine storage tanks (carbon steel)
30. Install amine filtration system for contaminant removal
31. Mount amine make-up system for losses
32. Install acid gas knockout drum
33. Connect acid gas compression system
34. Install sulfur dioxide analyzer on acid gas stream
35. Mount corrosion monitoring probes in amine system
36. Install amine reclaimer for thermal regeneration
37. Connect amine vapor recovery system
38. Install level and temperature controls throughout unit
39. Mount relief valves and flare connections
40. Install process instrumentation and transmitters

# Multilayer PCB Manufacturing Process

## Phase 1: Material Preparation and Incoming Inspection

1. Receive copper-clad laminate sheets in controlled humidity environment
2. Inspect laminate sheets for warpage using optical flatness measurement
3. Check copper thickness uniformity across sheet surface using X-ray fluorescence
4. Verify laminate dielectric constant at multiple frequencies
5. Measure sheet dimensions and compare against specifications
6. Inspect for surface defects including scratches, dents, and contamination
7. Store laminates in temperature-controlled room at 21°C ± 2°C
8. Receive prepreg sheets and inspect for resin content uniformity
9. Check prepreg thickness using micrometer at multiple points
10. Verify prepreg glass weave pattern and alignment
11. Test prepreg flow characteristics using differential scanning calorimetry
12. Store prepreg in freezer at -18°C to prevent premature curing
13. Receive photoresist materials and verify batch numbers
14. Check photoresist viscosity and shelf life
15. Store photoresist in dark, temperature-controlled environment

## Phase 2: Inner Layer Preparation

16. Remove inner layer copper-clad laminates from storage 24 hours before processing
17. Allow laminates to reach room temperature to prevent condensation
18. Cut laminate sheets to working panel size using precision shears
19. Create tooling holes in panel corners for registration alignment
20. Clean copper surface using mechanical scrubbing with pumice
21. Rinse panels with deionized water at 18 megohm-cm resistivity
22. Apply acidic micro-etch solution to roughen copper surface
23. Rinse panels thoroughly to remove all etchant residue
24. Dry panels using forced hot air at controlled temperature
25. Inspect copper surface for uniform matte finish
26. Apply oxide alternative treatment for improved lamination adhesion
27. Bake panels at 100°C for 15 minutes to remove moisture
28. Cool panels to room temperature in controlled environment

## Phase 3: Inner Layer Imaging

29. Load CAM data for inner layer circuit patterns into imaging system
30. Calibrate laser direct imaging (LDI) system for accuracy
31. Apply liquid photoresist using curtain coating method
32. Control photoresist thickness to 8-12 microns
33. Pre-bake photoresist at 80°C for 10 minutes in convection oven
34. Load panel into LDI system with precision alignment
35. Expose photoresist using 405nm UV laser at calculated dosage
36. Image circuit traces with 50-micron line/space resolution
37. Image ground planes with controlled copper removal areas
38. Create fine-pitch traces for high-speed signal routing
39. Image thermal relief patterns for component pads
40. Expose registration targets for layer alignment verification
41. Post-exposure bake panel at 100°C for 20 minutes
42. Develop photoresist using sodium carbonate solution
43. Spray develop solution at controlled pressure and temperature
44. Rinse developed panel with deionized water
45. Inspect imaged pattern using automated optical inspection (AOI)
46. Verify trace width, spacing, and registration accuracy
47. Check for photoresist residue, bridging, or missing features
48. Touch up minor defects using repair station if within tolerance

## Phase 4: Inner Layer Etching

49. Load panel into conveyorized etching system
50. Apply cupric chloride or alkaline ammonia etchant
51. Control etchant temperature at 48-52°C for optimal etch rate
52. Spray etchant at high pressure for uniform copper removal
53. Monitor etch rate using inline conductivity sensors
54. Remove exposed copper while protecting resist-covered areas
55. Achieve controlled undercut for designed trace profile
56. Rinse panel thoroughly with water to stop etching process
57. Strip photoresist using sodium hydroxide solution
58. Rinse panel with deionized water to remove all chemistry
59. Dry panel using hot air knife system
60. Inspect etched inner layer using AOI system
61. Verify trace width and spacing meet design specifications
62. Check for over-etching, under-etching, or copper residue
63. Measure copper thickness of etched traces using cross-section
64. Verify impedance test coupons are properly formed
65. Apply automated defect marking for rework identification
66. Store completed inner layers in protective packaging

## Phase 5: Lamination Stack-up Preparation

67. Retrieve prepreg sheets from freezer storage
68. Allow prepreg to thaw at room temperature for 4 hours
69. Cut prepreg sheets to panel size using precision cutting system
70. Count and organize prepreg sheets by resin content grade
71. Retrieve etched inner layers in correct stacking sequence
72. Clean inner layer surfaces using light brushing
73. Inspect inner layers for contamination or oxidation
74. Prepare mirror-finish stainless steel press plates
75. Clean press plates using lint-free wipes and IPA solution
76. Apply release film to press plates for easy separation
77. Place bottom press plate on lamination fixture
78. Position bottom copper foil for outer layer formation
79. Lay first prepreg sheet with proper fiber direction orientation
80. Align inner layer-2 using tooling pins for registration
81. Place prepreg sheets between inner layer-2 and core
82. Position central core laminate in stack-up
83. Add prepreg sheets above core laminate
84. Align inner layer-3 using tooling pins
85. Place final prepreg sheets over inner layer-3
86. Position top copper foil for outer layer formation
87. Add release film over top copper foil
88. Place top press plate to complete stack-up assembly
89. Verify stack-up sequence matches design specification
90. Check alignment of all tooling holes across stack-up
91. Record stack-up configuration and batch information

## Phase 6: Lamination Pressing

92. Load prepared stack-up into hydraulic vacuum press
93. Close press chamber and initiate vacuum cycle
94. Evacuate air to pressure below 1 torr
95. Apply initial pressure of 100 PSI to compress stack-up
96. Begin heating cycle with controlled ramp rate
97. Increase temperature to 180°C over 45 minutes
98. Monitor temperature using embedded thermocouples
99. Increase pressure to 300 PSI as resin begins to flow
100. Hold temperature at 180°C for 90 minutes for full cure
101. Monitor resin flow using edge squeeze-out observation
102. Maintain vacuum throughout pressing cycle
103. Begin controlled cooling at rate of 3°C per minute
104. Reduce pressure gradually during cooling phase
105. Cool to below 60°C before press opening
106. Release vacuum and open press chamber
107. Remove laminated panel from press
108. Inspect panel for proper bonding and resin bleed
109. Check panel thickness using precision micrometer
110. Verify panel flatness using surface measurement system
111. Trim excess material from panel edges
112. Bake panel at 150°C for 4 hours to complete cure
113. Store laminated panels in vertical racks to prevent warpage

## Phase 7: Drilling Operations

114. Load panel CAM drilling data into CNC drilling machine
115. Calibrate drill spindle runout to below 10 microns
116. Install micro-grain carbide drill bits in tool magazine
117. Set drill parameters for different hole sizes and depths
118. Stack multiple panels with aluminum entry and backing sheets
119. Secure stack using pin lamination tooling
120. Align stack using optical vision system
121. Drill pilot holes for registration verification
122. Drill through-hole vias at 25,000 RPM spindle speed
123. Use peck drilling cycle for aspect ratios above 10:1
124. Maintain drill bit sharpness by monitoring hit count
125. Replace drill bits after programmed number of holes
126. Drill microvias using UV laser ablation system
127. Set laser parameters for controlled depth drilling
128. Create blind vias from outer layer to inner layer
129. Drill buried vias within inner layer structures
130. Use CO2 laser for larger via sizes above 150 microns
131. Inspect drilled holes using automated vision inspection
132. Verify hole size, position accuracy, and wall quality
133. Check for burrs, smear, or drill breakage
134. Destack panels and remove entry/backing materials
135. Debur holes using mechanical or chemical methods

## Phase 8: Copper Plating and Metallization

136. Clean drilled panels using high-pressure water spray
137. Remove drill smear using permanganate desmear process
138. Apply desmear chemistry to etch resin from hole walls
139. Rinse panels thoroughly to remove all permanganate residue
140. Apply electroless copper deposition process
141. Clean and micro-etch copper surfaces for activation
142. Apply palladium catalyst to hole walls
143. Immerse panels in electroless copper bath at controlled pH
144. Deposit initial copper layer 0.5-1.0 micron thick
145. Verify electroless copper coverage using continuity testing
146. Load panels into electrolytic copper plating line
147. Apply copper sulfate electrolyte at controlled current density
148. Plate copper to 25-30 micron thickness on hole walls
149. Plate outer layer surfaces uniformly
150. Monitor plating thickness using inline measurement
151. Control plating parameters for uniform distribution
152. Rinse panels with deionized water after plating
153. Dry panels using forced air system
154. Measure copper thickness in holes using cross-section analysis
155. Verify copper coverage meets IPC Class 3 requirements

## Phase 9: Outer Layer Imaging and Etching

156. Apply photoresist to outer layer surfaces using roller coating
157. Control photoresist thickness to 15-20 microns
158. Soft bake photoresist at 80°C for 15 minutes
159. Load outer layer artwork into LDI imaging system
160. Expose photoresist using registered laser imaging
161. Image fine-pitch traces, pads, and routing patterns
162. Create solder mask defined (SMD) pad structures
163. Expose test coupons and quality control patterns
164. Post-exposure bake at 100°C for 20 minutes
165. Develop photoresist using conveyorized spray system
166. Inspect outer layer imaging using AOI
167. Etch exposed copper using cupric chloride etchant
168. Control outer layer etch to achieve designed trace profile
169. Strip photoresist after etching completion
170. Clean and dry panels after resist stripping

## Phase 10: Solder Mask Application

171. Apply liquid photoimageable solder mask using screen printing
172. Control solder mask thickness to 20-30 microns
173. Tack dry solder mask at 80°C for 10 minutes
174. Load solder mask artwork into LDI system
175. Expose solder mask openings for pads and vias
176. Develop solder mask using sodium carbonate solution
177. Inspect solder mask openings for accuracy and cleanliness
178. Thermal cure solder mask at 150°C for 60 minutes
179. Verify solder mask adhesion and coverage

## Phase 11: Surface Finish Application

180. Select ENIG (Electroless Nickel Immersion Gold) process
181. Clean exposed copper pads using micro-etch
182. Apply electroless nickel plating to 3-6 micron thickness
183. Immerse panels in gold plating bath for final finish
184. Deposit 0.05-0.1 micron gold over nickel layer
185. Rinse and dry panels after surface finish application
186. Inspect surface finish for uniformity and defects

## Phase 12: Routing and Final Processing

187. Load routing CAM data into CNC routing machine
188. Route panel outline and internal cutouts
189. Create breakaway tabs for panel depaneling
190. Bevel board edges at 45-degree angle

## Phase 13: Electrical Testing and Quality Control

191. Perform electrical continuity testing on all nets
192. Execute automated flying probe testing for opens and shorts
193. Verify impedance on controlled impedance traces
194. Conduct final dimensional inspection and measurement
195. Package finished multilayer PCBs for shipment to assembly facility

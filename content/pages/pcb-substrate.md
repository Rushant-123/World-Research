# PCB FR-4 Substrate Manufacturing

**Market Leader:** [Isola Group](/companies/isola-group)



## Raw Material Preparation

### Glass Cloth Preparation
1. Receive E-glass fabric rolls from glass cloth supplier in controlled storage area
2. Verify glass cloth specifications including thread count (typically 106-116 threads per inch)
3. Inspect fabric for defects including broken filaments, foreign particles, and weave irregularities
4. Measure fabric thickness using micrometer (typically 0.1-0.2mm for standard FR-4)
5. Check fabric weight per unit area (typically 35-300 g/m²)
6. Store glass cloth in climate-controlled room at 21°C ±2°C and 50% ±5% relative humidity
7. Unroll glass cloth on inspection tables with backlighting to check uniformity
8. Use optical microscope to verify filament diameter (5-13 microns for E-glass)
9. Test fabric tensile strength in warp and weft directions using universal testing machine
10. Measure fabric width to ensure consistency (typically 1040-1290mm)
11. Check for proper selvage edges to prevent fraying during processing
12. Verify glass cloth style designation (e.g., 106, 1080, 2116, 7628)
13. Scan fabric using automated vision systems to detect surface anomalies
14. Record lot numbers and traceability information in manufacturing execution system
15. Pre-clean fabric using gentle air blowing to remove loose particles

### Epoxy Resin System Formulation
16. Receive liquid epoxy resin in temperature-controlled storage (typically DGEBA type)
17. Verify resin viscosity using Brookfield viscometer at 25°C (typically 4000-12000 cPs)
18. Check resin epoxide equivalent weight (EEW) using chemical titration (typically 185-192)
19. Measure resin density using pycnometer (approximately 1.16 g/cm³)
20. Store hardener components separately in sealed containers to prevent moisture absorption
21. Weigh epoxy resin base in stainless steel mixing vessel using precision scale
22. Add dicyandiamide (DICY) hardener at 4-6% by weight for latent cure system
23. Incorporate 2-methylimidazole accelerator at 0.03-0.05% by weight
24. Add brominated epoxy resin for flame retardancy (20-30% by weight for standard FR-4)
25. Include antimony trioxide as flame retardant synergist (5-8% by weight)
26. Mix in phenolic hardeners for enhanced thermal performance (optional for high-Tg formulations)
27. Add coupling agents (typically 0.5-1.5% silane) to improve glass-resin adhesion
28. Incorporate fillers such as silica or alumina (optional, 10-40% for specific applications)
29. Add degassing agents to reduce void formation during lamination
30. Include flow control agents to optimize resin distribution during pressing
31. Mix formulation using high-shear planetary mixer at controlled temperature (40-50°C)
32. Maintain mixing speed at 60-120 RPM to prevent air entrainment
33. Continue mixing for 90-180 minutes until homogeneous dispersion achieved
34. Monitor mixing temperature continuously to prevent premature curing (keep below 60°C)
35. Use vacuum degassing during mixing to remove entrapped air bubbles
36. Sample resin mixture for quality control testing including gel time measurement
37. Measure mixed resin viscosity to ensure proper impregnation characteristics (target 500-2000 cPs)
38. Check resin gel time using hot plate method at 170°C (typically 90-180 seconds)
39. Verify flame retardancy by UL-94 vertical burn test on cured samples
40. Adjust formulation composition if quality parameters are outside specifications
41. Filter resin mixture through 100-mesh screen to remove any particles or agglomerates
42. Transfer filtered resin to impregnation tank with temperature control system
43. Record resin batch number and formulation details in quality management system

## Resin Impregnation Process

### Prepreg Manufacturing Setup
44. Clean and prepare horizontal treater line (impregnation machine)
45. Install glass cloth supply rolls on unwind stand with tension control system
46. Thread fabric through dancer roll system for continuous tension regulation
47. Set fabric tension to 5-15 N/cm width to prevent distortion
48. Program treater line speed based on fabric style (typically 5-30 meters/minute)
49. Heat resin impregnation tank to working temperature (40-60°C)
50. Verify resin level in impregnation pan is sufficient for continuous operation
51. Install doctor blades or metering rods to control resin pickup amount
52. Set dip tank configuration for single or multiple immersion passes
53. Calibrate resin flow rate to match line speed and desired resin content
54. Prepare vertical drying tower with multiple temperature zones
55. Set tower zone temperatures progressively from 120°C to 180°C (bottom to top)
56. Verify hot air circulation systems are functioning properly in each zone
57. Install thermocouples at multiple points to monitor temperature profiles
58. Prepare prepreg collection rolls at output end of treater line
59. Set up interleaf paper dispensing system to separate prepreg layers
60. Prepare quality control station for inline monitoring of resin content

### Impregnation Execution
61. Start treater line at slow speed for initial setup and verification
62. Feed glass cloth into first dip tank containing epoxy resin formulation
63. Ensure complete wetting of fabric as it passes through resin bath
64. Control immersion time by adjusting line speed (typically 10-30 seconds)
65. Monitor resin penetration through fabric thickness using online sensors
66. Use squeeze rollers to remove excess resin and achieve target resin content
67. Adjust nip pressure on squeeze rollers (typically 2-5 bar) for proper metering
68. Verify no resin-starved areas or dry spots in impregnated fabric
69. Pass impregnated fabric through second dip tank if additional resin loading needed
70. Monitor resin viscosity in tanks continuously and adjust temperature as needed
71. Add fresh resin to tanks as needed to maintain consistent level
72. Enter fabric into vertical drying tower at controlled speed
73. Advance material through progressively hotter temperature zones
74. Evaporate solvent (if present) in initial lower-temperature zones (100-120°C)
75. Begin B-stage curing in intermediate zones (140-160°C)
76. Complete B-staging in upper zones (170-180°C) to achieve tack-free surface
77. Monitor degree of cure using DSC samples from different tower positions
78. Control residence time in tower (typically 3-6 minutes total) to achieve proper B-stage
79. Measure prepreg exit temperature using infrared sensors (should be 40-60°C after cooling)
80. Apply interleaf paper to bottom surface of prepreg as it exits tower
81. Apply protective film or paper to top surface of prepreg
82. Wind prepreg onto collection rolls with controlled tension (3-8 N/cm)
83. Monitor winding tension to prevent prepreg elongation or compression
84. Record treater line operating parameters continuously in process control system
85. Take inline measurements of prepreg resin content using beta-ray gauge
86. Target resin content of 40-45% by weight for standard FR-4 prepreg
87. Sample prepreg periodically for offline quality testing
88. Measure prepreg thickness using micrometer (typically 0.09-0.18mm per ply)
89. Check prepreg tack by finger touch test (should be slightly tacky but not sticky)
90. Verify prepreg flow characteristics using parallel plate rheometer
91. Test prepreg gel time on hot plate at 170°C (should be 60-120 seconds)
92. Complete winding of prepreg roll when target length reached (typically 200-500 meters)
93. Remove completed roll from winder and seal in moisture-barrier bag
94. Label roll with batch number, production date, and expiration date
95. Store prepreg in freezer at -18°C to extend shelf life (typically 6 months frozen)

## Copper Foil Preparation

### Copper Foil Processing
96. Receive copper foil rolls from supplier (electrodeposited or rolled types)
97. Verify copper foil thickness using micrometer (typically 12-70 microns)
98. Check copper foil type designation (STD, RTF, VLP, or HVLP profile)
99. Inspect foil surface for pinholes, scratches, or contamination
100. Measure foil width to match laminate panel size requirements
101. Verify copper purity specifications (typically 99.8% minimum)
102. Check surface treatment on foil shiny side (typically nodular or oxide treatment)
103. Verify anti-tarnish treatment on foil matte side for adhesion promotion
104. Measure foil tensile strength using universal testing machine
105. Test foil elongation properties (typically 3-15% depending on type)
106. Clean foil surfaces using solvent wiping or plasma treatment if necessary
107. Cut foil to sheet size matching laminate panel dimensions using precision shear
108. Stack copper foil sheets with interleaf paper to prevent scratching
109. Store prepared foil sheets in clean room environment to prevent oxidation
110. Handle copper foil with gloves to prevent contamination from skin oils

## Lamination Process

### Lay-up Preparation
111. Retrieve prepreg rolls from freezer storage and allow to equilibrate to room temperature
112. Monitor prepreg warm-up time (typically 2-4 hours) before use to prevent condensation
113. Remove protective films from prepreg in clean room environment
114. Cut prepreg sheets to required panel size using computer-controlled cutter
115. Align prepreg grain direction (warp/weft) consistently for all plies
116. Clean lamination press platens using isopropyl alcohol
117. Install mirror-polished stainless steel press plates in lamination press
118. Apply release film on press plates to prevent resin adhesion
119. Place first copper foil sheet on press plate with treated side facing up
120. Position prepreg ply on copper foil ensuring complete coverage and no wrinkles
121. Align prepreg edges with copper foil edges for proper registration
122. Add additional prepreg plies according to stack-up design (typically 2-8 plies)
123. Ensure each prepreg ply is oriented consistently for uniform properties
124. Remove any air pockets by smoothing each ply during lay-up
125. Place second copper foil sheet on top of prepreg stack with treated side down
126. Add release film on top copper foil surface
127. Position second press plate on top of complete stack
128. Repeat lay-up process for multiple panels in multi-opening press
129. Verify all lay-ups are complete and properly aligned before press closure
130. Record lay-up configuration and material lot numbers for traceability

### Press Cycle Execution
131. Close lamination press and initiate automatic press cycle program
132. Apply initial contact pressure (0.5-1.5 MPa) to hold stack in place
133. Begin heating press platens using thermal oil, steam, or electric heating
134. Ramp platen temperature to 170-180°C at controlled rate (2-4°C/minute)
135. Monitor laminate internal temperature using thermocouples in dummy panels
136. Hold at lower pressure during heat-up phase to allow resin flow
137. Allow resin to melt and flow to fill voids when temperature reaches 140-160°C
138. Increase pressure gradually to final lamination pressure (1.5-3.5 MPa)
139. Apply full pressure when laminate temperature reaches 170°C
140. Monitor resin flow from panel edges to verify proper impregnation
141. Hold at full temperature and pressure for cure cycle (60-120 minutes)
142. Control press atmosphere using vacuum or nitrogen purge to prevent oxidation
143. Monitor degree of cure using real-time dielectric analysis if available
144. Begin cooling cycle after cure time completed, reducing temperature at controlled rate
145. Reduce pressure gradually during cooling to prevent delamination
146. Cool laminate to below glass transition temperature before opening press (typically below 100°C)
147. Monitor cooling rate to control internal stress development (typically 2-3°C/minute)
148. Open press when platen temperature reaches 60-80°C
149. Remove laminated panels from press using vacuum lifts to prevent handling damage
150. Inspect panel edges for proper resin flow and consolidation
151. Allow panels to cool to room temperature on flat cooling tables
152. Separate panels from release films and press plates

## Post-Lamination Processing

### Panel Conditioning
153. Stack laminated panels flat with weight on top to prevent warping during cooling
154. Allow panels to equilibrate at room temperature for minimum 24 hours
155. Trim panel edges using diamond saw or router to remove excess resin
156. Remove edge trim waste and recycle copper foil where possible
157. Clean panel surfaces using soft brushes or air jets to remove debris
158. Measure final laminate thickness using digital micrometer at multiple points
159. Verify thickness uniformity across panel (tolerance typically ±10%)
160. Record actual thickness values and compare to target specification
161. Inspect panels for surface defects including pits, voids, and resin starvation
162. Check for delamination by gentle flexing and visual inspection of edges
163. Look for blistering, warpage, or measling (white spots) indicating moisture or poor cure
164. Use ultrasonic C-scan to detect internal voids or delamination (for critical applications)

### Glass Transition Temperature (Tg) Control
165. Sample representative panels from each production lot for Tg measurement
166. Cut test coupons (approximately 5x5mm) from panel corners using precision saw
167. Clean test coupons to remove surface contamination
168. Prepare differential scanning calorimetry (DSC) instrument with nitrogen purge
169. Place sample in DSC aluminum pan with lid
170. Program DSC temperature ramp from 30°C to 250°C at 10°C/minute
171. Run DSC scan and analyze thermogram for glass transition inflection point
172. Measure Tg from midpoint of transition step in heat flow curve
173. Verify Tg meets specification (typically 130-140°C for standard FR-4, 170-180°C for high-Tg)
174. Perform dynamic mechanical analysis (DMA) for more precise Tg measurement if required
175. Measure Tg from peak in tan delta curve or onset of storage modulus drop
176. Verify degree of cure is complete (residual cure exotherm should be minimal)
177. If Tg is below specification, investigate press temperature, pressure, or time
178. Perform post-cure in oven at 160-180°C for 2-4 hours if additional cure needed
179. Re-measure Tg after post-cure to verify improvement

## Quality Testing and Characterization

### Electrical Testing
180. Cut test coupons for electrical property measurements per IPC-TM-650 methods
181. Measure dielectric constant at 1 MHz using impedance analyzer (typically 4.2-4.6 for FR-4)
182. Test dissipation factor at 1 MHz (typically 0.01-0.02 for standard FR-4)
183. Measure volume resistivity using high-resistance meter (typically >10^13 ohm-cm)
184. Test surface resistivity to verify insulation properties (typically >10^12 ohms)
185. Perform dielectric breakdown strength test using progressive voltage (typically >40 kV/mm)
186. Measure electrical properties at elevated temperature and humidity if required
187. Test insulation resistance after moisture conditioning per IPC-4101 requirements

### Mechanical and Thermal Testing
188. Measure flexural strength using three-point bend test (typically 415-450 MPa)
189. Test peel strength between copper foil and laminate (typically >1.0 N/mm)
190. Perform delamination test at elevated temperature (288°C for 10 seconds minimum)
191. Measure coefficient of thermal expansion (CTE) in X, Y, and Z directions
192. Verify Z-axis CTE is below specification (typically <70 ppm/°C below Tg)
193. Test flammability rating per UL-94 (verify V-0 rating for FR-4)
194. Perform final visual inspection and mark panels with lot identification codes
195. Package qualified FR-4 substrate panels for shipment to PCB fabricators with certificates of compliance

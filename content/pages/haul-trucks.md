## Manufacturing Process for 400-Ton Capacity Mining Haul Trucks

### Frame Fabrication and Structural Assembly (Steps 1-35)

1. Receive high-strength low-alloy steel plates (HSLA-80/100) in 50-150mm thickness for main frame rails
2. Inspect steel certificates for yield strength, tensile properties, and chemical composition compliance
3. Transfer steel plates to CNC plasma cutting station with water table for thermal distortion control
4. Load CAD files for frame rail profiles into CNC cutting system with nesting optimization
5. Perform CNC plasma cutting of main longitudinal frame rails (12-14 meters length)
6. Cut cross-member sections, gusset plates, and mounting brackets from plate steel inventory
7. Edge preparation by automated beveling machine for full-penetration welds (30-45 degree angles)
8. Transport cut components to shot blasting booth for surface preparation and rust removal
9. Apply weld-through primer coating to faying surfaces using HVLP spray equipment
10. Position main frame rails on precision alignment jig with laser measurement system
11. Install cross-members at engineered spacing (800-1200mm intervals) using temporary tack welds
12. Verify frame geometry using laser tracker with ±2mm tolerance across entire structure
13. Perform robotic GMAW welding of longitudinal seams with 100% penetration requirement
14. Execute manual FCAW welding of cross-member connections with certified welders
15. Install reinforcement gussets at high-stress zones identified by finite element analysis
16. Weld suspension mounting towers with precision alignment to wheelbase specifications
17. Add engine mounting rails with vibration isolation provisions and alignment features
18. Install dump body hoist mounting pedestals with spherical bearing pockets
19. Weld fuel tank support brackets and protective shielding to frame underside
20. Attach transmission mounting crossmember with precision-machined alignment surfaces
21. Install electrical conduit runs and protective channels along frame structure
22. Add hydraulic line routing brackets with cushioned clamps for vibration isolation
23. Perform 100% ultrasonic testing of critical welds per AWS D1.1 structural welding code
24. Conduct magnetic particle inspection of weld heat-affected zones for crack detection
25. Stress-relieve entire frame assembly in controlled atmosphere furnace at 595°C for 4 hours
26. Shot blast completed frame to SA 2.5 cleanliness standard for coating adhesion
27. Apply zinc-rich epoxy primer (75-100 microns DFT) by airless spray in paint booth
28. Cure primer coating at 60°C for 2 hours in convection oven
29. Apply polyurethane topcoat system (150-200 microns DFT) in two-tone manufacturer colors
30. Bake final coating at 80°C for 90 minutes for full cross-linking
31. Install frame identification plates with serial number, capacity rating, and manufacturing date
32. Mount ground connection points for electrical system with corrosion-resistant hardware
33. Install lifting lugs for in-service maintenance and component replacement procedures
34. Apply protective wax coating to internal frame cavities for corrosion prevention
35. Transfer completed frame to main assembly hall on specialized transport carriage

### Drivetrain Installation (Steps 36-65)

36. Receive V16 or V20 diesel engine (2500-4000 HP) from engine supplier with test documentation
37. Inspect engine for shipping damage and verify serial number against build specification
38. Install engine oil (400-600 liters) and coolant (600-800 liters) for pre-installation testing
39. Connect engine to dynamometer test stand for no-load operational verification
40. Remove shipping brackets and install production engine mounts with torque specifications
41. Position overhead crane with 50-ton capacity for engine lifting operation
42. Attach lifting sling to engine eyebolts with load balancing considerations
43. Lower engine assembly onto frame mounting rails with precision alignment protocol
44. Install vibration isolation mounts between engine and frame (8-12 mounting points)
45. Torque engine mounting bolts to 1200-1500 N⋅m with calibrated hydraulic wrenches
46. Connect engine wiring harness with weatherproof Deutsch connectors
47. Route high-pressure fuel lines from tanks to engine with fire-resistant sleeving
48. Install air intake system with multi-stage filtration and pre-cleaner cyclone
49. Connect exhaust system with flexible bellows and vertical discharge stack (6-8 meters)
50. Install exhaust after-treatment system for emissions compliance (SCR catalyst with DEF injection)
51. Mount radiator assembly (2.5-3.5 meter width) on hinged access frame for serviceability
52. Connect coolant hoses with constant-tension clamps rated for 120°C operating temperature
53. Install hydraulically-driven cooling fan (1.8-2.2 meter diameter) with variable speed control
54. Position electric drive alternator/generator (2500-3000 kVA) on engine-mounted platform
55. Install high-voltage cables (1000-1500V DC) from alternator to wheel motor controllers
56. Mount wheel motor control cabinets on frame with environmental sealing (IP65 rated)
57. Install regenerative braking resistor grids with forced air cooling systems
58. Connect wheel motors (800-1200 HP each) at rear axle with splined shaft couplings
59. Install planetary reduction gearboxes (30:1 to 40:1 ratio) at each wheel position

63. Install fuel tanks (3000-5000 liter capacity) with anti-slosh baffles and rollover protection

## Quality Assurance and Certifications

All mining haul trucks undergo rigorous testing to meet MSHA (Mine Safety and Health Administration) requirements in the USA or equivalent international mining safety standards. Key certifications include ROPS (Rollover Protective Structure) testing per ISO 3471, FOPS (Falling Object Protective Structure) per ISO 3449, and brake system performance per ISO 3450. Each truck is delivered with comprehensive documentation including weld inspection reports, dimensional verification records, load test results, and operator training materials. Manufacturing facilities maintain ISO 9001 quality management certification and implement statistical process control for critical dimensions and assembly procedures.

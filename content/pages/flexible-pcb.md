# Flexible Printed Circuit Manufacturing Process

**Market Leader:** [Flexium](/companies/flexium)



## Phase 1: Material Preparation and Inspection (Steps 1-15)

1. Receive polyimide film rolls in cleanroom environment
2. Inspect polyimide film for surface defects using automated optical inspection
3. Measure polyimide film thickness using micrometer (standard: 25μm ±2μm)
4. Check polyimide film moisture content using hygrometer
5. Condition polyimide film in controlled humidity chamber (50% RH, 23°C)
6. Unroll polyimide film onto inspection table
7. Remove protective liner from polyimide substrate
8. Clean polyimide surface using ionized air blower
9. Receive rolled copper foil shipment (18μm and 12μm thickness)
10. Inspect copper foil for oxidation and surface contamination
11. Measure copper foil thickness using X-ray fluorescence analyzer
12. Check copper grain structure under microscope
13. Store copper foil in nitrogen-filled storage cabinet
14. Prepare material traceability labels with batch numbers
15. Log all incoming material specifications in MES system

## Phase 2: Substrate Cleaning and Surface Treatment (Steps 16-30)

16. Load polyimide film onto pre-cleaning conveyor system
17. Apply alkaline cleaning solution to remove organic contamination
18. Rinse polyimide with deionized water (resistivity >18 MΩ·cm)
19. Apply micro-etching solution to roughen polyimide surface
20. Monitor surface roughness using atomic force microscopy
21. Rinse substrate with cascading DI water spray
22. Dry polyimide using infrared heating lamps (80°C)
23. Perform corona discharge treatment to enhance adhesion
24. Measure surface energy using contact angle goniometer
25. Apply silane coupling agent to polyimide surface
26. Bake substrate at 120°C for 15 minutes
27. Inspect treated surface for uniformity
28. Clean copper foil using sulfuric acid desmut process
29. Rinse copper foil thoroughly with DI water
30. Dry copper foil using nitrogen blower system

## Phase 3: Adhesive Application (Steps 31-45)

31. Prepare acrylic-based adhesive resin mixture
32. Filter adhesive through 5μm filter to remove particles
33. Adjust adhesive viscosity using dilution solvents
34. Load adhesive into precision coating system
35. Calibrate coating roller gap to 15μm
36. Apply adhesive layer to polyimide substrate using roll coating
37. Monitor adhesive thickness using laser displacement sensor
38. Inspect coating uniformity across substrate width
39. Pre-dry adhesive at 80°C for 5 minutes
40. Remove edge beads using precision trimming system
41. Apply second adhesive coat for enhanced bonding
42. Check adhesive tackiness using tack tester
43. Transport coated substrate through drying oven (100°C)
44. Measure residual solvent content using gas chromatography
45. Store semi-cured adhesive substrate in temperature-controlled area

## Phase 4: Copper Lamination Process (Steps 46-65)

46. Align polyimide substrate with copper foil using registration pins
47. Load substrate and copper into vacuum lamination press
48. Position release films on both sides of laminate stack
49. Close press chamber and initiate vacuum cycle (0.1 mbar)
50. Evacuate trapped air for 3 minutes
51. Heat press platens to 180°C
52. Monitor temperature uniformity across press surface
53. Apply lamination pressure (20 kg/cm²) gradually
54. Maintain pressure and temperature for 90 minutes
55. Monitor adhesive flow using pressure sensors
56. Check for delamination using ultrasonic inspection
57. Cool press to 60°C before opening
58. Remove laminated panel from press
59. Inspect copper-polyimide bond quality visually
60. Perform peel strength testing on sample coupons
61. Measure laminate thickness using digital micrometer
62. Check for wrinkles or air bubbles using backlight inspection
63. Trim laminate edges using precision shearing machine
64. Apply protective film to copper surface
65. Log lamination parameters in quality database

## Phase 5: Photoresist Application (Steps 66-80)

66. Load laminated panel into photoresist coating chamber
67. Clean copper surface using plasma treatment
68. Verify surface cleanliness using water break test
69. Dispense liquid photoresist onto panel center
70. Spin coat photoresist at 1500 rpm for 30 seconds
71. Achieve target photoresist thickness of 8μm
72. Soft bake photoresist at 90°C for 10 minutes
73. Cool panel to room temperature on chill plate
74. Apply dry film photoresist using hot roll laminator
75. Set laminator temperature to 110°C
76. Apply uniform pressure using rubber rollers
77. Remove protective liner from dry film
78. Inspect photoresist coating for defects
79. Measure photoresist thickness using stylus profiler
80. Store coated panels in dark, humidity-controlled room

## Phase 6: Photolithography and Exposure (Steps 81-100)

81. Clean photomask with acetone and isopropyl alcohol
82. Inspect photomask for defects using high-resolution microscope
83. Load photomask into mask aligner system
84. Position FPC panel on vacuum chuck
85. Align panel to photomask using alignment marks
86. Verify alignment accuracy (±10μm tolerance)
87. Close mask aligner and create hard contact
88. Set UV exposure dose to 150 mJ/cm²
89. Expose photoresist using mercury vapor lamp (365nm wavelength)
90. Monitor UV intensity with inline power meter
91. Complete exposure cycle (18 seconds)
92. Remove exposed panel from aligner
93. Post-exposure bake panel at 95°C for 5 minutes
94. Load panel into automatic spray developer
95. Develop exposed photoresist using alkaline developer
96. Monitor development progress using optical endpoint detection
97. Rinse developed panel with DI water
98. Dry panel using centrifugal dryer
99. Inspect circuit pattern using automated optical inspection
100. Touch up minor defects using laser repair station

## Phase 7: Copper Etching Process (Steps 101-115)

101. Load panel into horizontal spray etching machine
102. Pre-wet panel with water spray
103. Apply cupric chloride etchant at 45°C
104. Etch exposed copper using spray nozzle array
105. Monitor etch rate using conductivity sensors
106. Control etchant concentration (125 g/L copper)
107. Rinse etched panel with water to stop reaction
108. Inspect trace width and spacing using microscope
109. Measure conductor width (target: 75μm ±5μm)
110. Check for under-etching or over-etching
111. Strip photoresist using alkaline stripper
112. Rinse panel thoroughly with DI water cascade
113. Dry panel using hot air knife system
114. Inspect final circuit pattern with AOI system
115. Perform electrical continuity testing on test coupons

## Phase 8: Coverlay Preparation (Steps 116-130)

116. Receive coverlay film rolls with adhesive layer
117. Inspect coverlay for alignment holes and defects
118. Measure coverlay thickness (25μm polyimide + 15μm adhesive)
119. Cut coverlay to panel dimensions using laser cutter
120. Create openings for connector areas using CO2 laser
121. Verify opening dimensions match circuit design
122. Remove laser cutting debris using vacuum system
123. Inspect laser cut edges for carbonization
124. Clean coverlay surface with ionized air
125. Remove protective liner from adhesive side
126. Align coverlay to circuit pattern using registration pins
127. Check alignment accuracy using microscope
128. Prepare bonding press with release films
129. Set press temperature profile (180°C peak)
130. Verify press parallelism and pressure uniformity

## Phase 9: Coverlay Lamination (Steps 131-145)

131. Place circuit panel on press lower platen
132. Position aligned coverlay on circuit side
133. Add cushion pad to distribute pressure evenly
134. Apply top coverlay to bottom side of panel
135. Insert assembly into vacuum lamination press
136. Evacuate press chamber to 0.1 mbar
137. Heat press to 180°C over 20-minute ramp
138. Apply lamination pressure of 15 kg/cm²
139. Maintain bonding conditions for 60 minutes
140. Monitor adhesive melt flow optically
141. Cool press to 60°C with pressure maintained
142. Release pressure and open press chamber
143. Remove laminated panel from press
144. Inspect coverlay bond quality using peel test
145. Check for adhesive squeeze-out at edges

## Phase 10: Via Drilling and Formation (Steps 146-160)

146. Load panel into CNC drilling machine
147. Load 0.2mm micro drill bits into spindle
148. Calibrate drilling coordinates using fiducial marks
149. Set spindle speed to 80,000 rpm
150. Program drilling pattern for via holes
151. Execute drilling cycle with vacuum chip removal
152. Inspect via holes using automated vision system
153. Measure via diameter accuracy (±15μm tolerance)
154. Desmear drilled holes using permanganate solution
155. Remove drilling resin smear from via walls
156. Rinse panel with DI water thoroughly
157. Neutralize permanganate using reducing agent
158. Dry panel at 80°C for 10 minutes
159. Inspect via wall cleanliness using microscopy
160. Prepare panel for electroless copper deposition

## Phase 11: Via Metallization (Steps 161-175)

161. Activate via walls using palladium catalyst solution
162. Rinse panel to remove excess catalyst
163. Immerse panel in electroless copper bath
164. Deposit seed layer of copper (0.3μm thickness)
165. Monitor deposition rate using thickness probe
166. Rinse panel with DI water to stop reaction
167. Dry panel using nitrogen gas blower
168. Load panel into electroplating line
169. Apply electroplated copper to via walls (15μm)
170. Use pulsed current plating for better uniformity
171. Monitor plating thickness using X-ray fluorescence
172. Rinse plated panel with DI water
173. Perform cross-sectional analysis of via fill quality
174. Check via resistance using four-point probe
175. Verify electrical connection through via structures

## Phase 12: Surface Finish Application - ENIG (Steps 176-185)

176. Clean copper surface using micro-etch solution
177. Rinse panel with DI water cascade
178. Activate copper surface with pre-dip solution
179. Immerse panel in electroless nickel bath
180. Deposit nickel layer (3-5μm thickness) at 88°C
181. Monitor nickel deposition rate and uniformity
182. Rinse panel with DI water to remove nickel solution
183. Immerse panel in immersion gold bath
184. Deposit gold flash layer (0.05-0.1μm) at 85°C
185. Rinse panel with DI water and dry with nitrogen

## Phase 13: Stiffener Bonding (Steps 186-195)

186. Cut polyimide or FR-4 stiffener material to size
187. Apply acrylic adhesive to stiffener surface
188. Align stiffener to connector mounting areas
189. Position stiffener using alignment fixture
190. Bond stiffener using heated press (150°C, 5 kg/cm²)
191. Maintain bonding pressure for 30 minutes
192. Cool assembly and remove from press
193. Inspect stiffener bond strength using pull test
194. Perform final flexibility testing with bend radius measurement (minimum 2mm radius)
195. Conduct electrical testing, visual inspection, and package finished FPC for shipment to Mac display assembly facility

# Manufacturing Process: Steam Turbine Generator (500MW+)

## Phase 1: Rotor Forging and Machining (Steps 1-25)

1. Source high-grade chromium-molybdenum-vanadium steel ingot weighing 150 tons from specialty steel mill
2. Heat steel ingot to 1250°C in gas-fired furnace for 8 hours to achieve uniform temperature distribution
3. Transfer glowing ingot to 15,000-ton hydraulic forging press using overhead crane system
4. Perform initial upset forging to reduce height and increase diameter, applying 12,000 tons pressure
5. Rotate workpiece 90 degrees and perform cross-forging to refine grain structure
6. Execute draw-forging operations to elongate rotor blank to approximate final length of 12 meters
7. Use precision manipulators to rotate forging between press strokes for uniform deformation
8. Perform cogging operations with shaped dies to create rough rotor contour
9. Allow forged rotor to cool slowly in insulated pit for 72 hours to prevent thermal stress cracking
10. Transport rotor blank to heat treatment facility using specialized heavy-duty trailer
11. Perform normalizing heat treatment at 920°C for 6 hours followed by air cooling
12. Execute tempering at 650°C for 10 hours to achieve optimal strength-toughness balance
13. Conduct ultrasonic inspection of entire rotor volume to detect internal defects or inclusions
14. Mount rotor blank on heavy-duty lathe with 15-meter bed length and 3-meter swing capacity
15. Machine rotor journals to precise diameter tolerances of ±0.025mm using carbide cutting tools
16. Create blade attachment grooves using broaching machine with custom profile cutters
17. Machine balance grooves at both ends of rotor for future dynamic balancing operations
18. Mill axial keyways for coupling attachment using horizontal milling machine
19. Drill central bore through rotor axis for thermal stress relief and weight reduction
20. Bore and tap mounting holes for blade roots with precision thread specifications
21. Machine thrust collar surfaces to mirror finish for bearing contact
22. Create oil distribution grooves in journal surfaces for hydrodynamic lubrication
23. Shot-peen all machined surfaces to induce beneficial compressive residual stresses
24. Perform dimensional inspection using coordinate measuring machine and laser tracker
25. Apply temporary corrosion protection coating and transport to blade assembly area

## Phase 2: Turbine Blade Manufacturing and Attachment (Steps 26-50)

26. Cast high-temperature nickel-based superalloy blades using investment casting process
27. Create ceramic shell molds by repeatedly dipping wax patterns in ceramic slurry
28. Dewax molds in autoclave and fire at 1100°C to achieve high-strength ceramic shell
29. Pour molten superalloy at 1550°C into preheated molds under vacuum conditions
30. Allow castings to solidify directionally using controlled cooling to create columnar grain structure
31. Break away ceramic shell using mechanical vibration and chemical leaching
32. Cut off casting gates and runners using abrasive cutting wheel
33. Solution heat treat blades at 1200°C for 4 hours followed by rapid quenching
34. Age harden at 850°C for 24 hours to precipitate strengthening gamma-prime phase
35. Machine blade root attachment features to precise tolerances using 5-axis CNC machining
36. Grind blade airfoil surfaces to achieve required aerodynamic profile accuracy
37. Drill internal cooling passages using EDM drilling for small diameter precision holes
38. Apply thermal barrier coating to blade surfaces using air plasma spray process
39. Inspect each blade using fluorescent penetrant testing for surface crack detection
40. Measure blade weight and calculate moment values for rotor balancing calculations
41. Clean rotor blade attachment grooves using solvent and compressed air
42. Insert blade roots into axial-entry fir-tree slots in rotor rim
43. Secure blades using mechanical locking devices or expansion pins
44. Install inter-blade damping wires through holes near blade tips to reduce vibration
45. Weld or braze shroud segments between blade tips to form continuous ring
46. Verify blade attachment torque values using calibrated torque wrenches
47. Perform visual inspection of all blade installations for proper seating and alignment
48. Conduct tap testing of each blade to verify secure attachment via acoustic response
49. Apply identification markings to each blade for traceability and maintenance records
50. Cover completed blade row with protective wrapping for transport to balancing facility

## Phase 3: Generator Stator Manufacturing (Steps 51-75)

51. Fabricate stator frame from welded steel plates forming cylindrical structure 6 meters in diameter
52. Machine mounting feet and alignment surfaces on stator frame using large boring mill
53. Install radial cooling ducts and ventilation passages in stator frame structure
54. Stack electrical steel laminations (0.35mm thick silicon steel) to build stator core
55. Apply insulating coating between lamination layers to minimize eddy current losses
56. Compress lamination stack using hydraulic press to specified pressure
57. Install through-bolts and clamping structures to maintain lamination compression
58. Machine stator core inner diameter to precise roundness and concentricity specifications
59. Cut axial slots in stator core inner surface using precision milling machine
60. Insert slot cell insulation made from mica-epoxy composite material
61. Form copper bar conductors into precise shapes matching stator slot geometry
62. Apply ground insulation wrapping to each conductor bar using mica tape
63. Insert bottom layer conductor bars into stator slots using hydraulic pressing tools
64. Place slot separator insulation between upper and lower conductor layers
65. Insert top layer conductor bars to complete slot filling
66. Drive slot wedges to secure conductors and maintain tight packing
67. Connect conductor bars at stator ends to form series-parallel winding configuration
68. Braze or weld conductor connections using silver-bearing alloys
69. Insulate and support end-winding structures using glass-fiber reinforced epoxy brackets
70. Apply additional turn insulation and stress grading materials at connection points
71. Install thermocouples in stator windings for temperature monitoring during operation
72. Connect stator winding leads to terminal bushings mounted on stator frame
73. Perform winding resistance measurements to verify electrical continuity
74. Execute high-potential testing at 2x rated voltage plus 1000V to verify insulation integrity
75. Apply final protective coating to end windings and transport to generator assembly area

## Phase 4: Generator Rotor Manufacturing (Steps 76-100)

76. Forge generator rotor body from single steel ingot using process similar to turbine rotor
77. Machine rotor body to cylindrical form with precision journals for bearing support
78. Mill axial slots in rotor body surface to receive field winding conductors
79. Apply slot insulation using mica-based materials rated for operating temperatures
80. Wind copper field coils using rectangular cross-section conductors
81. Insulate field coils with multiple layers of mica tape and vacuum-pressure impregnation
82. Insert field coils into rotor body slots using hydraulic pressing equipment
83. Drive retaining wedges to secure field windings against centrifugal forces
84. Install non-magnetic retaining rings at rotor ends using shrink-fit process
85. Machine retaining ring outer surfaces concentric with rotor centerline
86. Install rotor shaft extensions and coupling flanges using shrink-fit assembly
87. Machine coupling bolt holes and alignment features with precision
88. Drill radial holes for rotor balancing weight adjustments
89. Install slip rings on rotor shaft for field current supply via brushes
90. Connect field winding ends to slip ring segments using flexible conductors
91. Apply insulation to all electrical connections on rotor surface
92. Perform electrical testing of field windings including resistance and high-potential tests
93. Balance generator rotor in low-speed balancing machine to reduce vibration
94. Mount rotor in high-speed balance test rig spinning at 110% rated speed
95. Measure vibration at multiple speeds and add/remove balance weights as needed
96. Verify rotor balance meets specification of less than 25 microns runout at rated speed
97. Perform overspeed test to 120% rated speed to verify mechanical integrity
98. Conduct magnetic field mapping to verify uniform flux distribution
99. Apply protective coating to rotor surfaces and install shaft end protectors
100. Transport balanced rotor to final generator assembly building using air-ride trailer

## Phase 5: Final Assembly (Steps 101-130)

101. Install lower half of turbine casing on foundation using precision alignment equipment
102. Position turbine rotor with attached blades into lower casing using overhead crane
103. Install turbine bearings and verify journal clearances using dial indicators
104. Connect turbine rotor to generator rotor using precision-machined coupling
105. Align coupling faces to within 0.05mm parallelism using laser alignment system
106. Install coupling bolts with hydraulic tensioning to specified preload values
107. Lower generator stator over generator rotor maintaining minimum air gap clearance
108. Position stator on foundation and align to within 0.1mm of design position
109. Install generator air gap sensors to monitor clearance during operation
110. Install upper half of turbine casing and torque casing bolts to specification
111. Install turbine diaphragms containing fixed blade rows between rotor stages
112. Verify axial clearances between rotating and stationary components using feeler gauges
113. Install turbine steam inlet connections and expansion joints
114. Connect steam extraction lines for feedwater heating system
115. Install turbine exhaust hood and condenser connection
116. Mount turbine front standard bearing housing and lubrication system
117. Install turbine rear bearing with self-aligning capability for thermal expansion
118. Connect generator stator cooling water supply and return piping
119. Install generator hydrogen cooling system components and piping
120. Connect generator excitation system including brushes and slip rings
121. Install protective enclosures around generator coupling and exciter
122. Mount main generator circuit breaker and isolated-phase bus connections
123. Install generator neutral grounding transformer and resistor
124. Connect generator potential transformers and current transformers for protection
125. Install turbine control system including governor actuators and valves
126. Mount turbine supervisory instrumentation including vibration probes and thermocouples
127. Connect lubrication oil supply and return lines to all bearing housings
128. Install oil cooling heat exchangers and filtration equipment
129. Connect control and monitoring wiring to central control system
130. Perform complete system walkdown inspection verifying all installations

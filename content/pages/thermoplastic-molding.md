# Thermoplastic Component Molding for Mac Components

## Material Preparation and Blending

1. Receive polycarbonate (PC) resin pellets in moisture-proof packaging
2. Verify PC resin grade specification matches Apple material requirements
3. Inspect resin lot number and traceability documentation
4. Check PC resin color consistency using spectrophotometer
5. Receive ABS (Acrylonitrile Butadiene Styrene) resin in sealed containers
6. Verify ABS grade matches impact resistance specifications
7. Document incoming material batch numbers in quality system
8. Store PC resin in climate-controlled warehouse at 20-25°C
9. Store ABS resin away from direct sunlight and moisture
10. Transfer PC resin to desiccant dryer hopper
11. Set PC drying temperature to 120°C with -40°C dew point
12. Dry PC resin for minimum 4 hours before processing
13. Transfer ABS resin to separate drying hopper
14. Set ABS drying temperature to 80-90°C
15. Dry ABS resin for minimum 2-3 hours
16. Monitor moisture content using in-line moisture analyzer
17. Ensure PC moisture level below 0.02% before molding
18. Ensure ABS moisture level below 0.05% before molding
19. Program gravimetric blender for 70% PC / 30% ABS ratio
20. Calibrate blender accuracy to ±0.5% tolerance
21. Load dried PC resin into blender hopper A
22. Load dried ABS resin into blender hopper B
23. Set blender throughput rate matching injection machine capacity
24. Initiate continuous blending cycle with mixing auger
25. Monitor blend ratio in real-time via control system
26. Collect blend samples every 2 hours for verification
27. Test blend samples for homogeneity using microscopy
28. Add impact modifier additives at 2% concentration if specified
29. Incorporate mold release agent at 0.3% concentration
30. Mix UV stabilizer additives for outdoor-exposed components
31. Add flame retardant additives for UL94 V-0 rating if required
32. Blend thermal stabilizers to prevent degradation during processing
33. Incorporate antistatic agents for electronic component housings
34. Mix colorants to achieve precise Pantone color match
35. Verify final blend composition with spectroscopic analysis

## Injection Molding Machine Setup

36. Select 450-ton hydraulic injection molding machine
37. Verify machine calibration certificates are current
38. Inspect machine barrel for wear or damage
39. Clean barrel interior with purging compound
40. Install 65mm diameter general-purpose screw
41. Verify screw compression ratio of 2.5:1 for PC/ABS
42. Check screw non-return valve operation
43. Install barrel heating bands in all five zones
44. Connect temperature controllers to barrel zones
45. Set rear zone temperature to 220°C
46. Set middle zones to progressive 235-245°C
47. Set front zone (nozzle) temperature to 250°C
48. Install barrel insulation jackets for thermal stability
49. Heat barrel to target temperatures over 45-minute ramp
50. Verify actual temperatures match setpoints within ±3°C
51. Purge barrel with sacrificial material to clear contaminants
52. Mount precision injection mold onto fixed platen
53. Align mold centerline with machine centerline
54. Secure mold with hydraulic clamps at all mounting points
55. Connect cooling lines to mold water channels
56. Install quick-disconnect fittings for cooling circuits
57. Connect hot runner system power cables
58. Wire hot runner thermocouples to controller
59. Set hot runner manifold temperature to 265°C
60. Set hot runner nozzles to 270°C for optimal flow
61. Heat hot runner system gradually over 30 minutes
62. Monitor hot runner temperature zones for uniformity
63. Install ejector pin return system
64. Connect hydraulic lines to core-pull mechanisms
65. Test mold opening and closing cycles without material
66. Verify mold protection settings prevent over-compression
67. Set clamp tonnage to 400 tons for cavity pressure
68. Adjust tie-bar parallelism for even mold compression
69. Install safety gates and light curtains around machine
70. Connect material feed line from blender to hopper

# Cable Assembly Machine

## Machine Setup and Initialization

1. Position the cable assembly machine on a level, vibration-free industrial floor surface.
2. Verify that the foundation can support the machine weight of approximately 850-1200 kg.
3. Connect the machine to a three-phase 400V AC power supply with proper grounding.
4. Install the main power cable through the rear access panel to the distribution box.
5. Connect compressed air supply line at 6-8 bar pressure to the pneumatic input port.
6. Install air filtration and regulation unit upstream of the machine connection point.
7. Mount the control cabinet securely to the machine frame using provided brackets.
8. Connect the PLC communication cable from the main controller to the HMI touchscreen.
9. Install safety light curtains around the machine perimeter according to safety standards.
10. Connect emergency stop buttons at multiple access points around the machine.
11. Mount the wire spool holder assembly to the rear of the machine frame.
12. Install tension control arms on the wire feeding path to maintain consistent wire tension.
13. Thread the wire through the straightening rollers to remove any coil set from spooled wire.
14. Adjust straightening roller pressure based on wire diameter and material properties.
15. Guide the wire through the measuring wheel encoder for accurate length measurement.

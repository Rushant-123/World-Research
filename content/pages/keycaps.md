## Material Selection and Preparation

1. Select ABS (Acrylonitrile Butadiene Styrene) plastic resin for standard keycaps with shore hardness 85-95A
2. Alternatively select PBT (Polybutylene Terephthalate) resin for premium keycaps with superior wear resistance
3. Choose resin color masterbatch matching target color specification with ΔE < 1.0
4. Verify resin lot number and material certification for FDA compliance if applicable
5. Inspect raw resin pellets for contamination, discoloration, or moisture damage
6. Transfer resin pellets from storage silo to hopper dryer using pneumatic conveying system
7. Load resin into desiccant hopper dryer maintaining -40°C dew point
8. Dry ABS resin at 80°C for minimum 4 hours to reduce moisture content below 0.05%
9. Dry PBT resin at 120°C for minimum 6 hours to reduce moisture content below 0.02%
10. Monitor moisture level using inline moisture analyzer with NIR spectroscopy
11. Verify drying completion by testing sample pellets for brittleness and surface appearance
12. Transfer dried resin to injection molding machine hopper with sealed hot air connection

## Injection Mold Setup

13. Select multi-cavity mold with 16-32 cavities per shot for high-volume production
14. Inspect mold surfaces for wear, scratches, or residue from previous production runs
15. Clean mold cavities using ultrasonic cleaning bath with alkaline detergent at 60°C
16. Rinse mold thoroughly with deionized water and dry with compressed air at 0.6 MPa
17. Apply mold release agent (silicone-based) to cavity surfaces if using PBT material
18. Install mold into injection molding machine with 150-350 ton clamping force
19. Connect mold cooling channels to chilled water circulation system at 15-25°C
20. Install hot runner system maintaining melt temperature at 230-250°C for ABS
21. Calibrate ejector pins for proper keycap release with 3-5mm travel distance
22. Set up slide cores for undercut features (Windows key logos, specialized profiles)
23. Install texture inserts for desired surface finish (matte, glossy, or custom texture)
24. Verify mold alignment and parallelism within 0.02mm tolerance

## Injection Molding Process

25. Set barrel temperature zones: rear 200°C, middle 220°C, front 240°C for ABS
26. For PBT keycaps, increase temperatures: rear 240°C, middle 260°C, front 270°C
27. Set injection pressure to 80-120 MPa depending on keycap size and wall thickness
28. Configure injection speed at 50-100 mm/s for optimal cavity filling
29. Set holding pressure at 60-80% of injection pressure for 3-5 seconds
30. Program cooling time of 15-25 seconds based on keycap wall thickness (1.2-1.8mm)
31. Set mold open time of 2-3 seconds for ejector pin actuation and part removal
32. Verify screw rotation speed at 60-100 RPM for consistent melt homogeneity
33. Set back pressure to 5-10 MPa during plasticizing phase
34. Initiate first shot and inspect for short shots, flash, or sink marks
35. Adjust injection parameters iteratively until keycaps meet dimensional specifications
36. Monitor melt cushion maintaining 3-5mm material reserve in front of screw
37. Verify gate freeze time ensuring proper packing phase completion
38. Run production cycle continuously at 12-18 second cycle time per shot

## Keycap Profile Formation

39. Produce cylindrical profile (most common) with concave top surface radius 8-12mm
40. Alternatively produce sculpted profile with varied row heights (R1-R4) for ergonomics
41. Form keycap wall thickness of 1.2mm minimum for structural integrity
42. Create stem socket cavity with 5.5mm x 4.0mm dimensions for Cherry MX compatibility
43. Form internal ribs (0.5mm thick) for reinforcement without increasing weight
44. Mold stabilizer mounting holes (2mm diameter) for larger keys (spacebar, shift, enter)
45. Ensure stem socket interior dimensions: 5.45-5.55mm length, 3.95-4.05mm width
46. Verify stem socket depth of 3.8-4.2mm for proper switch engagement
47. Create draft angles of 1-3° on vertical walls for easy mold release
48. Form LED window cavity (optional) for backlit keycaps with 0.8mm wall thickness
49. Produce double-shot compatible bases with first color forming legend area
50. Inspect keycap top surface flatness within 0.1mm across entire surface

## Surface Texture Application

51. Apply Type A texture (smooth matte) using EDM (Electrical Discharge Machining) on mold
52. Alternatively apply Type B texture (fine grain) with VDI 18-24 finish specification
53. Create custom texture patterns using laser etching on mold cavity surface
54. For glossy finish, polish mold cavity to mirror finish Ra < 0.1μm
55. Apply soft-touch coating post-molding by spraying polyurethane elastomer
56. Verify surface roughness Ra value: 0.8-1.6μm for standard matte finish
57. Measure gloss level at 60° angle: 5-15 GU for matte, 80-95 GU for glossy
58. Ensure consistent texture across all keycap positions and mold cavities
59. Apply anti-fingerprint coating (oleophobic) via vacuum deposition if specified
60. Inspect texture uniformity using raking light at 15° angle

## Legend Printing - Pad Printing Method

61. Design legend artwork with 0.5mm minimum feature size for clarity
62. Create photopolymer printing plate (cliché) using UV exposure and washout
63. Prepare UV-curable ink matching color specification (typically white or black)
64. Load ink into sealed ink cup maintaining consistent viscosity at 23°C
65. Install printing plate onto pad printing machine with registration accuracy ±0.1mm
66. Mount keycaps onto custom fixture holding 50-100 pieces simultaneously
67. Program pad printing cycle: ink pickup, transfer, cure with 2 second intervals
68. Execute first print stroke with silicone pad compressing 1-2mm onto keycap surface
69. Apply UV cure immediately after printing using 395nm LED array at 2 W/cm²
70. Achieve cure time of 0.5-1 second with mercury lamp or LED-UV system
71. Verify legend opacity using densitometer reading >1.5 for white on black
72. Inspect registration accuracy ensuring legend centered within ±0.2mm
73. Test ink adhesion using cross-hatch tape test (5B rating required)
74. Measure legend thickness of 8-15μm using non-contact profilometer

## Legend Printing - Laser Etching Method

75. Load molded keycaps onto CNC-controlled fixture with vacuum hold-down
76. Install fiber laser system (20-30W) with 1064nm wavelength for plastic marking
77. Program laser path tracing legend outline with 0.05mm spot size
78. Set laser power to 60-80% for surface ablation without material charring
79. Configure scan speed at 800-1200 mm/s for optimal marking quality
80. Set pulse frequency to 20-30 kHz for fine detail reproduction
81. Apply first laser pass creating shallow engraving depth of 0.1-0.15mm
82. Execute multiple passes (2-3) if deeper contrast required for dark keycaps
83. Purge ablated material using compressed air nozzle during marking process
84. Verify legend contrast by visual inspection under 500 lux illumination
85. Measure engraving depth consistency across batch using confocal microscope
86. Fill laser-etched legend with contrasting paint using tampo pad if color required
87. Cure infill paint at 80°C for 30 minutes in convection oven
88. Clean excess paint from keycap surface using IPA (isopropyl alcohol) wipe

## Legend Printing - Double-Shot Molding Method

89. Produce first shot keycap base using opaque colored plastic (typically dark color)
90. Design first shot with legend area as through-holes or recessed cavities
91. Transfer first shot keycaps to second mold using robotic picker with vacuum cups
92. Load second color plastic (typically white or translucent) for legend fill
93. Inject second shot filling legend cavities with perfect registration
94. Allow second shot to bond chemically with first shot during cooling phase
95. Achieve molecular bonding between ABS/ABS or PBT/PBT material combinations
96. Verify legend will never fade or wear off due to integrated construction
97. Inspect legend edge definition ensuring sharp boundaries between colors
98. Measure legend protrusion/recession relative to keycap surface: ±0.05mm
99. Verify backlight transmission if translucent second shot used for RGB illumination
100. Test legend adhesion by attempting physical separation (should be impossible)

## Legend Printing - Dye Sublimation Method

101. Select PBT keycaps as substrate (required for dye sublimation process)
102. Design legend artwork with high-resolution vector graphics for clarity
103. Print legend onto transfer paper using dye-sublimation printer with disperse dyes
104. Load PBT keycaps onto heat press fixture with precision alignment pins
105. Position printed transfer paper over keycap surface with legend aligned
106. Apply silicone rubber pad over transfer paper for uniform pressure distribution
107. Close heat press applying 0.3-0.5 MPa pressure at 180-200°C temperature
108. Hold press closed for 60-90 seconds allowing dye to sublimate and penetrate PBT
109. Open press and remove transfer paper revealing permanently dyed legend
110. Verify dye penetration depth of 50-100μm below surface for wear resistance
111. Inspect color accuracy comparing against Pantone reference with ΔE < 2.0
112. Test legend durability by rubbing with 1000 grit sandpaper (minimal fading)
113. Measure color fastness to light using xenon arc lamp exposure (500+ hours)
114. Package dye-sublimated keycaps without additional coating or protection needed

## UV Clear Coating Application

115. Load printed keycaps onto conveyor belt feeding UV coating line
116. Clean keycap surfaces using ionized air blower removing static and dust
117. Apply UV-curable clear coat via spray application at 15-20 psi air pressure
118. Maintain coating thickness of 10-15μm for protection without affecting feel
119. Pass coated keycaps through UV tunnel with mercury vapor lamps at 120 W/cm
120. Achieve full cure in 2-3 seconds with conveyor speed of 5-8 m/min
121. Alternatively use LED-UV system at 395nm wavelength for faster cure
122. Verify coating hardness of 3-4H pencil hardness after cure
123. Inspect coating uniformity ensuring no runs, sags, or orange peel texture
124. Measure coating thickness using non-destructive ultrasonic gauge
125. Test coating adhesion using cross-hatch tape test (minimum 4B rating)
126. Verify coating does not yellow after 1000 hours UV aging test

## Dimensional Inspection

127. Remove keycaps from injection molding machine using robotic arm or manual picking
128. Allow keycaps to cool to room temperature (23°C) for dimensional stability
129. Measure overall keycap height using digital caliper: R1=11.5mm, R2=10.5mm, R3=9.5mm, R4=8.5mm
130. Verify keycap top surface dimensions: 12.7mm x 12.7mm for standard 1U key
131. Measure stem socket position centered within ±0.15mm of keycap center
132. Check stem socket internal dimensions using go/no-go gauge: 5.5mm x 4.0mm
133. Verify wall thickness uniformity measuring at 4 points: 1.2-1.8mm range
134. Inspect keycap base flatness placing on granite surface plate (within 0.1mm)
135. Measure stabilizer bar slot dimensions: 13.0-13.2mm spacing for 2U keys
136. Verify draft angle on vertical surfaces measuring 1-3° using angle gauge
137. Check for warpage by measuring diagonal dimensions (difference < 0.2mm)
138. Inspect gate vestige height ensuring < 0.1mm protrusion for smooth mounting

## Stem Compatibility Testing

139. Select sample Cherry MX switches (Red, Brown, Blue) for compatibility testing
140. Install keycap onto switch stem using gentle downward pressure
141. Verify keycap seats fully with stem socket bottoming against switch housing
142. Measure keycap height above PCB surface: 10.5-11.5mm for standard profile
143. Test keycap wobble by applying lateral force < 0.5mm movement in any direction
144. Verify stem grip force: 5-8N insertion force, 3-5N removal force
145. Cycle keycap installation/removal 100 times checking for stem socket wear
146. Test compatibility with alternative switch types: Gateron, Kailh, Outemu
147. Verify keycap remains securely attached during vigorous typing simulation
148. Inspect stem socket for cracking or stress marks after removal
149. Test interference with adjacent keycaps in standard ANSI layout (0.5-1mm clearance)
150. Verify keycap allows full switch travel (4mm actuation) without binding

## Stabilizer Compatibility Testing

151. Install 2U keycap (Backspace) onto plate-mounted stabilizer assembly
152. Verify stabilizer stems insert smoothly into keycap mounting slots
153. Measure stabilizer stem socket spacing: 12.0mm, 24.0mm, or 50.0mm as applicable
154. Test keycap level by measuring height difference at both ends < 0.2mm
155. Apply force to keycap corners verifying balanced actuation without binding
156. Install 6.25U spacebar checking stabilizer wire clearance underneath
157. Verify spacebar remains level during full keystroke cycle (±0.3mm)
158. Test for rattling noise by typing rapidly on stabilized keys
159. Inspect stabilizer stem socket for proper depth: 4.5-5.0mm
160. Verify keycap does not contact stabilizer wire during bottom-out
161. Check for adequate clearance between stabilizer stems and keycap walls (0.3mm minimum)
162. Test Cherry-style vs Costar-style stabilizer compatibility as required

## Color Matching and Consistency

163. Select color standard reference chips from Pantone or RAL color system
164. Place keycaps under D65 daylight simulator at 6500K color temperature
165. Compare keycap color against reference chip using visual assessment
166. Measure color using spectrophotometer recording L*a*b* values
167. Calculate color difference ΔE between sample and standard (target ΔE < 1.0)
168. Verify color consistency across different mold cavities (ΔE < 0.5 within batch)
169. Check color matching between keycap sets from different production runs
170. Monitor color drift over extended production by measuring every 1000 shots
171. Adjust masterbatch concentration if color deviation exceeds tolerance
172. Verify UV-printed legends match specified color with densitometer reading
173. Test color stability by exposing to 500 hours xenon arc lamp aging
174. Document color formulation and batch traceability for quality records

## Surface Roughness and Finish Quality

175. Measure surface roughness Ra using portable profilometer at 5 locations per keycap
176. Verify matte finish Ra value: 0.8-1.6μm for comfortable tactile feel
177. Measure gloss at 60° angle using glossmeter: target 8-15 GU for matte
178. Inspect surface for visual defects: sink marks, flow lines, or gate blush
179. Check for die lines or mold imperfections visible under raking light
180. Verify texture consistency comparing keycaps from different cavities
181. Test fingerprint resistance by handling with oily fingers and assessing visibility
182. Measure static friction coefficient: 0.3-0.5 for comfortable typing feel
183. Verify coating uniformity on UV-coated keycaps using microscope at 50x magnification
184. Inspect legend edges for sharpness and cleanliness (no bleeding or feathering)
185. Test wearability by rubbing with standardized abrasion tester (1000 cycles minimum)

## Final Quality Control and Packaging

186. Perform 100% visual inspection of all keycaps under bright LED illumination
187. Sort keycaps by keyboard layout position (alphas, modifiers, spacebars, etc)
188. Reject defective keycaps showing flash, short shots, discoloration, or legend defects
189. Package keycaps in protective plastic bags preventing scratching during shipping
190. Include desiccant packet if high humidity environment during storage/shipping
191. Label package with keycap profile, color, legend method, and compatibility information
192. Perform random sampling: measure 10 keycaps per 1000 for dimensional verification
193. Conduct drop test on packaged keycaps ensuring no damage from 1m height
194. Prepare certificates of compliance documenting material composition and testing
195. Ship keycaps to keyboard assembly facility or end customers via logistics partner

# Precision Measurement Tools Manufacturing Process

**Market Leader:** [Mitutoyo](/companies/mitutoyo)



## I. Material Preparation and Quality Control (Steps 1-15)

### Raw Material Inspection
1. Receive stainless steel bar stock (SUS420J2 martensitic grade) with material certificates
2. Verify chemical composition using optical emission spectrometry (OES) for carbon content 0.26-0.40%
3. Inspect chromium content (12.0-14.0%) and verify corrosion resistance specifications
4. Measure hardness of raw material using Rockwell hardness tester (HRC 50-55 after treatment)
5. Check dimensional tolerances of bar stock using coordinate measuring machine (CMM)
6. Inspect surface finish of raw material (Ra < 0.8 μm) using surface roughness tester
7. Verify straightness of bar stock using precision granite surface plate and dial indicators
8. Document material traceability with heat lot numbers and supplier certifications
9. Quarantine non-conforming materials and initiate supplier corrective action requests
10. Store approved materials in climate-controlled warehouse (20°C ± 2°C, 45% RH)

### Measuring Scale Preparation
11. Receive precision glass scales with photolithographic graduations (0.01mm intervals)
12. Inspect scale graduation accuracy using interferometric measurement system
13. Verify scale flatness using optical flat and monochromatic light source (sodium lamp)
14. Check thermal expansion coefficient of glass substrate (α < 8 × 10⁻⁶/°C)
15. Clean scales in ultrasonic bath with isopropyl alcohol and dry in laminar flow cabinet

## II. Caliper Body Manufacturing (Steps 16-40)

### Machining Main Beam
16. Cut stainless steel bar to rough length using precision band saw with coolant
17. Face both ends of bar using CNC lathe to establish reference surfaces (flatness 0.002mm)
18. Machine sliding groove using 5-axis CNC machining center with carbide end mills
19. Achieve groove surface finish of Ra 0.2 μm for smooth slider movement
20. Mill reference edge using grinding operation to achieve straightness of 0.005mm per 300mm
21. Drill mounting holes for scale attachment using precision jig boring machine
22. Ream mounting holes to H7 tolerance (±0.006mm for 6mm diameter)
23. Machine datum face using surface grinding with aluminum oxide wheel (46 grit)
24. Lap datum face using cast iron lap plate with diamond compound (1 μm grit)
25. Achieve final flatness of 0.001mm over entire datum surface

### Slider Component Manufacturing
26. Machine slider body from solid stainless steel block using 3-axis CNC mill
27. Create precision sliding surfaces with parallelism tolerance of 0.003mm
28. Mill vernier scale mounting slot with depth tolerance of ±0.01mm
29. Drill and tap adjustment screw holes (M3 × 0.5 pitch) for friction control
30. Machine jaw interfaces with perpendicularity tolerance of 0.005mm to sliding axis
31. Deburr all edges using precision hand files and fine abrasive paper (600 grit)
32. Polish sliding surfaces using magnetic abrasive finishing for Ra 0.1 μm
33. Apply hard chrome plating (5-10 μm thickness) for wear resistance
34. Lap chrome-plated surfaces to final dimensional tolerance (±0.002mm)
35. Ultrasonic clean slider in alkaline cleaner followed by deionized water rinse

### Jaw Manufacturing
36. Machine measuring jaws from hardened tool steel (JIS SKS3, HRC 58-62)
37. Grind measuring faces using precision surface grinder with CBN wheel
38. Achieve measuring face flatness of 0.0005mm and parallelism of 0.001mm
39. Hone measuring edges to 0.01mm radius using diamond honing stones
40. Apply titanium nitride (TiN) coating via PVD process for hardness HV 2300

## III. Micrometer Component Manufacturing (Steps 41-70)

### Anvil and Spindle Production
41. Machine anvil from tungsten carbide blanks using diamond grinding wheels
42. Achieve anvil measuring face flatness of 0.0002mm using lapping process
43. Grind spindle from hardened steel rod (JIS SUJ2, HRC 60-64)
44. Turn spindle thread using precision CNC lathe (0.5mm pitch, class 4h tolerance)
45. Grind spindle measuring face to spherical form (R > 100mm) or flat configuration
46. Lap spindle face to surface finish of Ra 0.05 μm using diamond paste
47. Measure spindle thread pitch using thread pitch comparator (accuracy 0.0001mm)
48. Inspect thread flank angle using optical projection comparator (tolerance ±30 arc-seconds)
49. Apply hard chrome plating to spindle (10-15 μm) for corrosion protection
50. Final grind spindle to nominal diameter (6.5mm ±0.002mm) after plating

### Frame and Sleeve Manufacturing
51. Cast micrometer frame using gray cast iron (FC200) for thermal stability
52. Age castings for 6 months at ambient conditions to relieve residual stresses
53. Machine internal thread in spindle bore using single-point boring tool
54. Grind internal thread to 6H tolerance class using threaded grinding wheel
55. Machine anvil mounting interface with perpendicularity tolerance 0.002mm
56. Lap frame measuring faces using precision lapping plate with 3 μm diamond compound
57. Drill holes for thimble bearing using precision jig borer (tolerance ±0.001mm)
58. Ream bearing bores to H6 tolerance for press-fit bearing installation
59. Machine sleeve with external thread matching spindle pitch (0.5mm)
60. Engrave graduation marks on sleeve using precision CNC engraving machine

### Thimble and Ratchet Mechanism
61. Machine thimble from stainless steel tube using CNC Swiss-type lathe
62. Create graduation marks (0.01mm resolution) using laser engraving system
63. Verify graduation spacing accuracy using vision measurement system (tolerance ±0.005mm)
64. Machine internal bore for spindle clearance fit (H8/f7 tolerance)
65. Manufacture ratchet spring from spring steel (SUS301-CSP, hardness HV 450)
66. Coil ratchet spring to specified force (5-10N) using CNC spring coiler
67. Heat treat ratchet spring at 350°C for stress relief without hardness loss
68. Machine ratchet stop mechanism with adjustable torque setting (5-15 Nm)
69. Polish thimble exterior to mirror finish (Ra < 0.05 μm) using buffing compound
70. Apply anti-reflective coating to graduation area for improved readability

## IV. Digital Display System Manufacturing (Steps 71-95)

### Linear Encoder Production
71. Manufacture glass scale carrier from low-expansion glass ceramic (Zerodur)
72. Deposit chromium metal layer (100nm thickness) via vacuum sputtering
73. Apply photoresist coating using spin-coating process (uniform 2 μm thickness)
74. Expose graduation pattern using laser interferometer lithography system
75. Develop photoresist using specified developer chemistry and time control
76. Etch chromium layer using wet chemical etching (ceric ammonium nitrate solution)
77. Strip residual photoresist using oxygen plasma ashing process
78. Inspect graduation quality using automated optical inspection (AOI) system
79. Apply protective coating (SiO₂, 200nm) via plasma-enhanced chemical vapor deposition
80. Mount scale to substrate using UV-curable adhesive with controlled thickness (20 μm)

### Photodetector Assembly
81. Fabricate silicon photodiode arrays using CMOS fabrication process
82. Wire-bond photodiodes to flexible printed circuit using gold wire (25 μm diameter)
83. Encapsulate wire bonds using transparent silicone potting compound
84. Attach infrared LED light sources (λ = 880nm) for transmission-type encoder
85. Align photodetector assembly to scale using precision X-Y-Z micropositioner
86. Achieve alignment accuracy of ±5 μm for optimal signal amplitude
87. Secure detector assembly using UV-curable adhesive and verify signal quality
88. Connect flex circuit to signal processing board using zero-insertion-force connector
89. Test encoder signal quality using oscilloscope (verify sinusoidal signals, 90° phase shift)
90. Calibrate encoder subdivision electronics for 0.01mm resolution output

### Digital Display Module
91. Manufacture liquid crystal display (LCD) with custom segment pattern
92. Program microcontroller with measurement algorithms and signal processing firmware
93. Integrate analog-to-digital converter (24-bit resolution) for encoder signal processing
94. Implement digital filtering using moving average and Kalman filter algorithms
95. Assemble display module with battery compartment (SR44 silver oxide battery)

## V. Coordinate Measuring Machine (CMM) Probe System (Steps 96-125)

### Touch Probe Mechanism
96. Machine probe body from titanium alloy (Ti-6Al-4V) for low mass and high stiffness
97. Drill central bore for stylus mounting using precision gun drill (tolerance ±0.005mm)
98. Thread bore using precision tap (M2 × 0.4 pitch, class 6H tolerance)
99. Manufacture kinematic coupling seats using precision grinding and EDM
100. Create three-point kinematic mount with carbide seats (120° spacing)
101. Lap kinematic seats to spherical form (R = 4mm) with 0.0002mm form accuracy
102. Manufacture corresponding kinematic balls from tungsten carbide (grade 25)
103. Grind balls to grade 5 tolerance (±0.00013mm diameter variation)
104. Assemble pre-load spring mechanism providing 0.5-1.0N seating force
105. Install strain gauge sensors for touch detection (sensitivity 0.01N)

### Stylus Manufacturing
106. Produce ruby balls using flame fusion crystal growth process (synthetic corundum)
107. Grind ruby spheres to precision grade 5 (diameter tolerance ±0.00013mm)
108. Measure sphere diameter using interferometric sphere gauge (uncertainty 0.00005mm)
109. Machine stylus stems from tungsten carbide rod using centerless grinding
110. Thread stem ends using precision roll-threading process (M2 × 0.4)
111. Adhesively bond ruby ball to stem using high-strength epoxy (shear strength > 50 MPa)
112. Cure adhesive joint at 150°C for 4 hours under controlled atmosphere
113. Inspect ball-to-stem concentricity using air bearing spindle rotary stage (< 0.001mm runout)
114. Qualify stylus effective working length using master artifact on CMM
115. Measure stylus tip position in 25 approach directions for probe compensation data

### Electronic Signal Processing
116. Design printed circuit board with low-noise amplifier for strain gauge signals
117. Implement Wheatstone bridge configuration with temperature compensation resistors
118. Integrate 16-bit analog-to-digital converter with 100 kHz sampling rate
119. Program FPGA with digital signal processing algorithms for trigger detection
120. Implement adaptive threshold algorithm compensating for probe speed variation
121. Design wireless transmission module (2.4 GHz ISM band) for cordless probe
122. Integrate lithium-ion battery (3.7V, 500mAh) with charging circuit
123. Program power management system for 40-hour continuous operation per charge
124. Implement encrypted communication protocol to prevent signal interference
125. Calibrate electronic gain and offset for each strain gauge channel

## VI. Gauge Block Manufacturing (Steps 126-145)

### Raw Material Preparation
126. Select ultra-stable tool steel (JIS SKS31) or tungsten carbide for gauge blocks
127. Verify material composition using X-ray fluorescence (XRF) spectroscopy
128. Measure material thermal expansion coefficient using dilatometer (-10 to +50°C range)
129. Document α = 11.5 × 10⁻⁶/°C ± 0.5 × 10⁻⁶/°C for steel grade
130. Rough machine blocks to oversized dimensions (+0.5mm) using surface grinder

### Precision Grinding and Lapping
131. Heat treat gauge blocks to HRC 60-65 using controlled atmosphere furnace
132. Temper at 180°C for stress relief maintaining hardness HRC 63 minimum
133. Stabilization heat treatment cycle: -80°C for 8 hours, then +100°C for 40 hours
134. Second stabilization cycle after rough grinding to relieve machining stresses
135. Precision grind blocks to near-final size (+0.001mm) using reciprocating surface grinder
136. Use fine-grit aluminum oxide wheel (220 grit, vitrified bond) with controlled infeed
137. Lap measuring faces using cast iron lap plate with diamond compound (9 μm, 3 μm, 1 μm progression)
138. Achieve flatness of 0.00005mm (50nm) for grade K gauge blocks
139. Lap faces to final length using ceramic lap plate with 0.25 μm diamond suspension
140. Control lapping temperature at 20°C ± 0.2°C using thermostatic room

### Measurement and Calibration
141. Pre-condition gauge blocks in metrology lab at 20°C ± 0.1°C for 24 hours
142. Measure length using laser interferometer (HeNe laser, λ = 632.991nm)
143. Correct for refractive index of air using Edlen equation with environmental parameters
144. Measure air temperature (±0.01°C), pressure (±10 Pa), humidity (±5% RH)
145. Achieve measurement uncertainty of 0.00002mm (20nm) for blocks up to 100mm

156. Verify vernier scale alignment with main scale using optical comparator (magnification 50×)

### Digital System Integration

## VIII. Micrometer Assembly Process (Steps 166-180)

### Precision Assembly
166. Press-fit bearings into frame thimble bore using arbor press with controlled force
167. Install spindle through frame and thread into sleeve with anti-backlash nut
168. Adjust spindle pre-load to eliminate axial play while maintaining smooth rotation
169. Install thimble over spindle with H8/f7 clearance fit allowing free rotation
170. Connect thimble to spindle using key or pin to transfer rotational motion
171. Install ratchet mechanism ensuring consistent measurement force application
172. Calibrate ratchet to slip at 5-10N force using calibrated force gauge
173. Install lock nut mechanism for holding measurement readings
174. Mount anvil to frame using precision ground shoulder screw (perpendicularity 0.002mm)

### Zero Setting and Calibration
175. Clean all measuring surfaces using lint-free cloth and ethanol
176. Bring measuring faces into contact using ratchet mechanism
177. Verify zero reading or measure zero error using gauge blocks
178. Adjust zero position using sleeve rotation or electronic offset if digital version
179. Test measurement at several points using grade 1 gauge block set
180. Document calibration results including errors at 0mm, 5mm, 10mm, 15mm, 20mm, 25mm

190. Calculate probe tip position offsets in X, Y, Z directions (uncertainty < 0.0005mm)

193. Verify probe repeatability using 10 repeated measurements (std dev < 0.0003mm)
194. Document probe qualification results including stylus effective diameter
195. Establish traceability chain to NIST through calibrated reference sphere certification

## Quality Assurance and Traceability

All measurement tools are calibrated against NIST-traceable standards with documented uncertainty budgets. Calibration certificates include measurement conditions (temperature, humidity, pressure), equipment used, and uncertainty calculations per ISO/IEC 17025. Tools are marked with unique serial numbers and calibration due dates. Digital measurement systems include built-in self-test routines and statistical process control for ongoing accuracy verification.

Measurement uncertainty for precision tools:
- Vernier calipers: ±0.02mm (k=2, 95% confidence)
- Digital calipers: ±0.01mm (k=2, 95% confidence)
- Micrometers: ±0.001mm (k=2, 95% confidence)
- CMM with touch probe: ±0.002mm (k=2, 95% confidence)
- Gauge blocks (grade K): ±0.00002mm (k=2, 95% confidence)

Environmental control requirements: All precision measurement operations performed in climate-controlled metrology laboratory maintained at 20°C ± 0.5°C (ISO 1 standard temperature) with humidity control at 45% ± 10% RH. Vibration isolation using pneumatic isolation tables for sub-micron measurements. Granite surface plates calibrated to grade AA flatness specification per ASME B89.3.7.

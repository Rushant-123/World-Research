# Speaker Spider Damper Manufacturing Process

**Market Leader:** [Precision Acoustics](/companies/precision-acoustics)



## Raw Material Preparation and Inspection

1. Receive rolls of cotton fabric from textile supplier in climate-controlled warehouse
2. Inspect fabric rolls for consistent thread count (typically 100-200 threads per inch)
3. Verify fabric weight specifications (usually 3-6 oz per square yard)
4. Check fabric for uniformity in weave pattern and tension
5. Examine fabric surface for contamination, oil spots, or foreign materials
6. Measure fabric moisture content using calibrated hygrometer (target 6-8%)
7. Store fabric in temperature-controlled environment at 68-72°F and 45-55% relative humidity
8. Allow fabric to acclimate for minimum 24 hours before processing
9. Unroll fabric samples and inspect for consistent width (tolerance ±0.5mm)
10. Test fabric tensile strength in both warp and weft directions
11. Verify fabric elongation properties meet specifications (typically 3-5% at break)
12. Check fabric for dimensional stability under heat exposure
13. Inspect fabric edges for fraying or irregular selvage
14. Document lot numbers and traceability information for quality control
15. Receive phenolic resin shipment in sealed containers

## Phenolic Resin Preparation

16. Verify phenolic resin batch number and manufacturing date
17. Check resin viscosity using Brookfield viscometer (target 200-400 cps)
18. Measure resin solids content using gravimetric analysis (target 40-50%)
19. Test resin pH level to ensure proper curing characteristics (pH 7-9)
20. Filter resin through 100-micron filter to remove particulates
21. Prepare resin mixing tank with stainless steel agitator
22. Heat resin to optimal processing temperature (75-85°F)
23. Add curing catalyst at specified ratio (typically 2-4% by weight)
24. Mix catalyst thoroughly for 15-20 minutes at controlled speed
25. Add flow control agents to optimize fabric penetration
26. Incorporate anti-foam additives to prevent bubble formation
27. Blend in plasticizers to control final spider compliance
28. Add UV stabilizers to prevent degradation over product lifetime
29. Mix fungicide and mildewcide agents for environmental resistance
30. Stir resin formulation for 30-45 minutes ensuring homogeneity
31. Check resin temperature stability during mixing process
32. Measure final resin viscosity after additive incorporation
33. Test resin gel time using heated test sample (target 8-12 minutes at 300°F)
34. Filter prepared resin again through fine mesh screen
35. Transfer resin to impregnation system reservoir

## Fabric Cutting and Preparation

36. Calculate required fabric dimensions based on spider diameter specifications
37. Program CNC cutting table with precise cutting patterns
38. Load fabric roll onto unwinding stand with tension control
39. Feed fabric through edge alignment system
40. Activate electrostatic elimination bars to prevent static cling
41. Execute automated cutting operation to create fabric blanks
42. Stack cut fabric pieces with interleaving paper to prevent sticking
43. Inspect cut edges for clean, uniform cuts without fraying
44. Count and batch fabric blanks for production lot control
45. Label batches with cutting date, operator ID, and lot number
46. Store cut fabric in sealed containers until impregnation
47. Verify fabric blank dimensions using calibrated measuring tools
48. Check for proper fiber orientation alignment in cut pieces
49. Remove any loose threads or debris from cut edges
50. Prepare fabric surface cleaning station with vacuum system

## Fabric Impregnation Process

51. Set up continuous impregnation line with multiple processing zones
52. Install fabric blanks onto precision feed rollers
53. Adjust feed roller tension to prevent fabric distortion
54. Pre-heat fabric blanks to 100-120°F to improve resin absorption
55. Feed fabric into resin bath at controlled rate (2-4 feet per minute)
56. Ensure complete fabric submersion in phenolic resin bath
57. Control resin bath temperature at 80-90°F for optimal viscosity
58. Allow fabric to dwell in resin for 30-60 seconds
59. Monitor resin penetration through fabric thickness using visual inspection
60. Pass impregnated fabric through precision squeeze rollers
61. Adjust roller gap to achieve target resin pickup (80-120% by weight)
62. Apply controlled pressure to remove excess resin
63. Collect excess resin in return tray for recycling
64. Monitor resin pickup weight using in-line scale system
65. Adjust squeeze roller pressure based on real-time measurements
66. Pass fabric through secondary resin distribution rollers
67. Ensure uniform resin distribution across fabric width
68. Check for dry spots or resin-rich areas
69. Apply heated air knife to remove surface excess resin
70. Verify uniform resin coating on both fabric surfaces

## Forming Die Preparation

71. Clean forming dies with approved solvent and lint-free cloths
72. Inspect die surfaces for damage, scratches, or contamination
73. Check die corrugation profile using precision measuring tools
74. Verify die dimensions match product specifications
75. Apply release agent coating to both male and female die surfaces
76. Buff release agent to thin, uniform coating
77. Allow release agent to cure for specified time (typically 10-15 minutes)
78. Pre-heat forming dies to operating temperature (280-320°F)
79. Monitor die temperature using calibrated thermocouples
80. Verify uniform temperature distribution across die surfaces
81. Check die alignment and parallelism using dial indicators
82. Adjust hydraulic press parameters for proper force application
83. Set press closing speed to prevent fabric displacement
84. Program press dwell time based on fabric thickness (typically 60-90 seconds)
85. Calibrate press force measurement system

## Fabric Forming and Molding

86. Position impregnated fabric blank onto lower forming die
87. Center fabric precisely using alignment fixtures
88. Ensure fabric edges extend beyond corrugation area
89. Activate press closing sequence at controlled rate
90. Apply initial forming pressure (500-800 psi) gradually
91. Allow resin to flow into corrugation profiles
92. Increase pressure to final molding force (1000-1500 psi)
93. Monitor die temperature throughout forming cycle
94. Maintain forming pressure during heat cure cycle
95. Hold part under pressure for 60-120 seconds at temperature
96. Allow phenolic resin to cross-link and cure partially
97. Monitor cure progress using embedded temperature sensors
98. Check for proper corrugation formation through visual inspection
99. Ensure uniform corrugation depth across spider diameter
100. Verify no fabric wrinkling or bunching has occurred

## Curing and Heat Treatment

101. Release hydraulic press pressure gradually to prevent part distortion
102. Open press and remove formed spider from die surface
103. Inspect spider for complete release from die without tearing
104. Transfer hot spider to post-cure oven using handling fixtures
105. Load spiders onto oven racks with proper spacing for air circulation
106. Close oven door and initiate post-cure heat cycle
107. Ramp oven temperature from ambient to 300°F over 30 minutes
108. Hold at 300°F for 60 minutes for initial cure stage
109. Increase temperature to 350°F over 15 minutes
110. Maintain 350°F for 90 minutes for full cure completion
111. Monitor oven atmosphere to prevent oxidation
112. Control air circulation rate for uniform heating
113. Record time-temperature profile for quality documentation
114. Allow oven to cool gradually to 200°F over 45 minutes
115. Remove spiders from oven when temperature reaches safe handling level
116. Transfer parts to cooling racks in climate-controlled area
117. Allow spiders to cool to room temperature naturally (typically 2-3 hours)
118. Monitor spider dimensions during cooling to detect shrinkage
119. Check for warping or dimensional changes during cool-down
120. Store cooled spiders in controlled environment before trimming

## Trimming and Edge Finishing

121. Load cured spiders into precision trimming fixture
122. Center spider using alignment pins or optical centering system
123. Verify spider orientation for consistent trimming results
124. Program CNC trimming machine with exact diameter specifications
125. Select appropriate cutting tool (rotary blade or laser system)
126. Execute automated trimming cycle to remove excess material
127. Trim outer diameter to final dimension (tolerance ±0.1mm)
128. Cut inner diameter mounting hole with precision tooling
129. Ensure clean, smooth cut edges without delamination
130. Remove trimming waste material and collect for disposal
131. Inspect trimmed edges for burrs or rough spots
132. Apply edge sanding operation if required for smoothness
133. Vacuum away dust and debris from trimming operation
134. Check final spider dimensions using calibrated measurement tools
135. Verify concentricity of inner and outer diameters

138. Check corrugation profile for uniform depth and spacing
139. Inspect for fabric show-through or insufficient resin coverage

141. Verify corrugations are symmetrical and evenly formed
142. Check for discoloration indicating improper cure
143. Examine edges for complete trimming and smooth finish
144. Inspect mounting area for proper formation and flatness
145. Look for contamination, foreign particles, or surface defects
146. Verify spider color consistency within acceptable range
147. Check for proper release agent residue removal
148. Examine corrugation peaks for sharpness and definition
149. Inspect valleys between corrugations for complete formation
150. Document any visual defects with digital photography

## Dimensional Verification

154. Measure corrugation amplitude (peak-to-valley distance)

156. Check concentricity between inner and outer diameters
157. Measure flatness of mounting flange area

## Compliance Testing

166. Install spider test sample in compliance measurement fixture
167. Mount fixture in universal testing machine with calibrated load cell
168. Apply zero load and record initial spider position
169. Program testing machine for displacement-controlled cycle
170. Apply gradually increasing force to spider center point
171. Measure force and displacement in both compression and extension
172. Record force-displacement curve over specified travel range (typically ±5mm)
173. Calculate compliance coefficient from slope of force-displacement curve
174. Verify compliance falls within specified range (typically 0.5-2.0 mm/N)
175. Check for linearity in force-displacement relationship
176. Measure hysteresis between loading and unloading curves
177. Verify symmetric compliance in both positive and negative directions
178. Test spider at multiple frequencies if dynamic compliance is specified
179. Record peak displacement and recovery characteristics
180. Check for permanent set after cyclic loading

## Linearity and Centering Force Measurement

181. Install spider in precision centering force test fixture
182. Apply radial offset displacement to simulate voice coil misalignment
183. Measure restoring force at various offset positions
184. Verify centering force increases proportionally with displacement
185. Calculate centering force constant (N/mm of offset)
186. Test centering force at multiple radial angles (0°, 90°, 180°, 270°)
187. Verify centering force symmetry around full circumference
188. Check that centering force meets minimum specification for alignment
189. Measure force linearity deviation over operating range
190. Test spider performance at elevated temperature (typical 150°F)
191. Verify compliance and centering force stability with temperature
192. Perform accelerated aging test on sample spiders (1000 cycles)
193. Re-measure compliance after aging to verify durability
194. Document all test results in quality assurance database
195. Package approved spiders in protective containers with lot identification for shipment to speaker assembly operations

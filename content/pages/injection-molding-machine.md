# Injection Molding Machine Manufacturing Process

## Machine Specifications and Setup (Steps 1-25)

1. Select Arburg Allrounder injection molding machine platform for Mac component production
2. Determine machine tonnage requirements based on part size and projected area
3. Evaluate 50-ton machines for small precision components (ports, buttons)
4. Assess 150-ton machines for medium components (bezels, internal brackets)
5. Configure 300-ton machines for large housings and enclosure panels
6. Specify 500-ton machines for MacBook Pro unibody plastic elements
7. Calculate clamping force: tonnage = projected area × material pressure factor
8. Verify platen dimensions accommodate maximum mold size (400mm × 400mm)
9. Configure tie bar spacing for mold installation and removal clearance
10. Set daylight opening distance for adequate mold access (minimum 500mm)
11. Specify injection unit size based on shot weight requirements
12. Calculate shot capacity: 40-60% of barrel capacity for optimal plastication
13. Select screw diameter: 28mm for small parts, 45mm for medium, 70mm for large
14. Determine L/D ratio (length to diameter): 20:1 to 24:1 for standard thermoplastics
15. Configure reciprocating screw design with compression ratio 2.5:1 to 3.5:1
16. Install non-return valve at screw tip to prevent material backflow
17. Set up barrel heating zones: rear, center, front, and nozzle
18. Install band heaters with individual temperature control per zone
19. Configure barrel cooling system with water channels for temperature stability
20. Mount hydraulic system with pump capacity matched to machine tonnage
21. Install oil reservoir with minimum 200-liter capacity and filtration system
22. Set up electrical control cabinet with Selogica control system

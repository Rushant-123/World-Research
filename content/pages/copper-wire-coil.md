# Speaker Voice Coil Manufacturing Process

**Market Leader:** [Foster Electric](/companies/foster-electric)



## Raw Material Preparation (Steps 1-15)

1. Receive copper magnet wire spools in AWG 38-42 gauge range from supplier
2. Inspect wire spool packaging for damage or contamination during transit
3. Verify wire gauge using precision micrometer (target tolerance ±0.001mm)
4. Check wire insulation integrity using continuity tester at 500V DC
5. Measure wire tensile strength using universal testing machine (minimum 15kg breaking load)
6. Verify copper purity certification (minimum 99.95% OFHC copper required)
7. Test insulation temperature rating (polyurethane-nylon coating rated to 180°C)
8. Inspect wire surface for scratches, nicks, or coating defects under magnification
9. Measure wire diameter at 10 points along 100m length for consistency
10. Record batch number and material certification in quality control database
11. Store wire spools in climate-controlled environment (20°C, 45% RH)
12. Load approved wire spools onto winding machine wire feeding mechanism
13. Thread wire through tension control system and guide eyelets
14. Set initial wire tension to 15-20 grams using spring-loaded tensioner
15. Verify wire path alignment through all guide points before winding

## Coil Former Preparation (Steps 16-30)

16. Receive polyimide coil formers from injection molding department
17. Inspect former outer diameter (target 12.5mm ±0.05mm tolerance)
18. Verify former wall thickness using ultrasonic thickness gauge (0.15mm nominal)
19. Check former height specification (8.2mm for Mac speaker application)
20. Inspect former surface finish for mold defects or contamination
21. Measure former roundness using coordinate measuring machine (maximum 0.02mm runout)
22. Test polyimide material temperature resistance (continuous operation to 260°C)
23. Verify former flange dimensions for wire retention during winding
24. Clean former surfaces using isopropyl alcohol and lint-free wipes
25. Dry formers in controlled oven at 80°C for 30 minutes to remove moisture
26. Apply primer coating to former exterior surface for improved adhesion
27. Cure primer at 120°C for 15 minutes in conveyor oven
28. Cool formers to room temperature in desiccator cabinet
29. Mount former onto precision mandrel of winding machine spindle
30. Secure former with pneumatic chuck at 2.5 bar clamping pressure

## Winding Machine Setup (Steps 31-50)

31. Power up CNC winding machine and initialize control system
32. Load voice coil winding program for Mac speaker specification
33. Set target turn count to 185 turns for this coil design
34. Program wire traverse pitch to 0.042mm per turn for tight winding
35. Configure spindle rotation speed to 1200 RPM for AWG 40 wire
36. Set wire tension control to 18 grams with ±2 gram tolerance
37. Adjust traverse guide positioning servo for precise wire placement
38. Calibrate laser turn counter for accurate layer monitoring
39. Set up adhesive dispensing system with thermal-cure epoxy
40. Prime adhesive pump and verify flow rate (0.05ml per layer)
41. Install wire break detection sensor with automatic stop function
42. Configure tension variation alarm limits (±3 grams triggers warning)
43. Set up vision system for wire placement quality monitoring
44. Calibrate resistance measurement probes for in-process testing
45. Initialize data logging system for process parameter recording
46. Perform test wind on dummy former to verify all parameters
47. Inspect test coil for proper wire spacing and adhesive coverage
48. Adjust any parameters based on test wind results
49. Clear test coil and prepare for production winding cycle
50. Reset turn counter and position traverse guide at start position

## Layer 1 Winding (Steps 51-65)

51. Manually attach wire end to former anchor point with temporary adhesive
52. Initiate automated winding cycle with start button
53. Monitor wire tension during first 10 turns (should stabilize at 18±1 grams)
54. Verify wire placement accuracy using vision system (±0.01mm target)
55. Check traverse increment per revolution matches programmed pitch
56. Observe wire seating on former surface for proper adhesion
57. Monitor spindle speed stability (±10 RPM maximum variation)
58. Complete first 47 turns comprising Layer 1 of coil winding
59. Apply first layer adhesive coating via micro-dispensing nozzle
60. Ensure uniform adhesive distribution across entire wire layer
61. Monitor adhesive viscosity (target 5000 cP at 25°C)
62. Verify adhesive penetration between wire turns using microscopy
63. Continue rotation for 30 seconds to distribute adhesive evenly
64. Measure layer 1 outer diameter (12.58mm expected)
65. Record Layer 1 completion time and all process parameters

## Layer 2 Winding (Steps 66-78)

66. Begin Layer 2 winding immediately after adhesive application
67. Maintain consistent 18-gram wire tension throughout layer
68. Monitor wire crossing angle over Layer 1 (should be perpendicular)
69. Verify proper nesting of Layer 2 wire in Layer 1 valleys
70. Complete 46 turns for Layer 2 (slightly fewer due to diameter increase)
71. Check for any wire tension spikes indicating catching or snagging
72. Apply Layer 2 adhesive coating using same dispensing parameters
73. Verify adhesive does not create voids or air pockets
74. Measure cumulative coil diameter after Layer 2 (12.66mm expected)
75. Inspect wire alignment using 40x magnification
76. Check for any layer-to-layer electrical shorts using megohmmeter
77. Record Layer 2 resistance increase (approximately 0.8 ohms added)
78. Allow 15 seconds for adhesive to partially set before continuing

## Layer 3 Winding (Steps 79-90)

79. Initiate Layer 3 winding sequence with proper tension reset
80. Monitor wire feed rate for consistency (12.5 meters/minute nominal)
81. Complete 46 turns for Layer 3 with precise turn counting
82. Apply Layer 3 adhesive with increased coverage for stability
83. Check cumulative turn count (139 turns completed at this point)
84. Measure coil diameter progression (12.74mm after Layer 3)
85. Verify no wire bunching or spacing irregularities present
86. Test coil resistance in-process (approximately 2.4 ohms measured)
87. Monitor adhesive cure state using infrared temperature sensor
88. Check former integrity under winding tension load
89. Record all Layer 3 parameters in quality database
90. Prepare for final layer winding with parameter verification

## Layer 4 Winding - Final Layer (Steps 91-105)

91. Begin Layer 4 (final layer) winding with increased monitoring
92. Complete remaining 46 turns to reach total 185 turn count
93. Apply final adhesive coating with enhanced coverage
94. Verify final coil outer diameter (12.82mm target dimension)
95. Monitor final wire tension to prevent last-turn loosening
96. Slow spindle speed for final 5 turns to improve precision
97. Cut wire using automatic cutter leaving 15mm tail length
98. Secure wire tail to coil body with quick-cure adhesive dot
99. Measure final DC resistance (3.2 ohms ±0.1 ohm specification)
100. Inspect complete coil for uniform appearance and wire coverage
101. Check coil height remains within 8.2mm ±0.1mm specification
102. Verify no wire crossovers or layer mixing occurred
103. Test coil-to-former bond strength by gentle manual pull test
104. Remove coil assembly from winding mandrel carefully
105. Place completed wound coil in curing rack for thermal processing

## Adhesive Curing Process (Steps 106-120)

106. Transfer wound coils to thermal cure oven on aluminum trays
107. Set oven temperature profile: ramp to 150°C over 20 minutes
108. Hold at 150°C for 45 minutes for complete epoxy cure
109. Monitor oven temperature uniformity (±3°C maximum variation)
110. Ensure adequate air circulation around each coil during cure
111. Monitor coil dimensional stability during thermal exposure
112. Check for any adhesive outgassing or bubbling defects
113. Complete thermal cure cycle with controlled cool-down phase
114. Reduce temperature at 2°C per minute to prevent thermal shock
115. Hold at 80°C for 15 minutes for stress relief
116. Cool to room temperature in desiccator environment
117. Inspect cured coils for any adhesive discoloration or cracking
118. Verify final bond strength exceeds 5N pull force requirement
119. Check that adhesive has not wicked beyond coil winding area
120. Record cure cycle parameters and oven calibration data

## Lead Wire Attachment (Steps 121-135)

121. Prepare tinned copper lead wires (AWG 34, 50mm length)
122. Strip lead wire insulation 2mm from each end using precision stripper
123. Clean wire ends with isopropyl alcohol to remove oxidation
124. Apply flux to wire ends for improved solder wetting
125. Tin lead wire ends with 63/37 tin-lead solder
126. Identify coil start and finish wire ends using continuity tester
127. Position coil in soldering fixture with lead access ports
128. Wrap coil start wire around first lead wire terminal 3 times
129. Solder first connection using 350°C iron with 1.5 second dwell
130. Inspect solder joint for complete wetting and fillet formation
131. Wrap coil finish wire around second lead wire terminal
132. Solder second connection with same temperature and time parameters
133. Verify both solder joints have no cold solder or poor contact
134. Test electrical continuity through complete coil and lead assembly
135. Measure total resistance including leads (3.25 ohms expected)

## Electrical Testing (Steps 136-155)

136. Connect coil to precision LCR meter for parameter measurement
137. Measure DC resistance at 1kHz, 1V test signal (3.2 ohm nominal)
138. Record resistance value with 0.01 ohm resolution
139. Verify resistance falls within 3.1-3.3 ohm acceptance range
140. Measure coil inductance at 1kHz test frequency
141. Record inductance value (target 0.25 mH ±0.02 mH)
142. Calculate coil Q factor from resistance and inductance values
143. Verify Q factor exceeds minimum specification of 2.5 at 1kHz
144. Perform impedance sweep from 20Hz to 20kHz frequency range
145. Record impedance magnitude and phase across frequency spectrum
146. Identify resonant frequency peak (should be above 25kHz)
147. Measure coil capacitance (typically 15-20 pF parasitic)
148. Test insulation resistance using 500V DC megohmmeter
149. Verify insulation resistance exceeds 100 megohms minimum
150. Perform hipot test at 1000V AC for 1 second duration
151. Check for any dielectric breakdown or leakage current
152. Measure coil temperature coefficient of resistance
153. Test coil at -20°C, 25°C, and 85°C temperature points
154. Calculate TCR value (typically +3900 ppm/°C for copper)
155. Record all electrical parameters in test database with pass/fail status

## Dimensional Verification (Steps 156-170)

156. Transfer coil to coordinate measuring machine for dimensional inspection
157. Measure coil outer diameter at 4 angular positions (0°, 90°, 180°, 270°)
158. Calculate average diameter and verify against 12.82mm ±0.05mm specification
159. Measure coil height from former base to wire surface
160. Verify height dimension of 8.2mm ±0.1mm is maintained
161. Check coil concentricity relative to former inner diameter
162. Measure radial runout (maximum 0.03mm allowed)
163. Verify former flange dimensions remain undistorted from winding
164. Measure wire pitch consistency across each layer
165. Check for any layer thickness variations exceeding 0.02mm
166. Inspect end turn positioning for proper wire termination
167. Measure lead wire attachment point locations for symmetry
168. Verify no wire protrusions beyond coil envelope dimensions
169. Check coil weight (target 0.85 grams ±0.05 grams)
170. Record all dimensional measurements with CMM certification data

## Thermal Performance Testing (Steps 171-185)

171. Mount coil in thermal test fixture with temperature monitoring
172. Attach thermocouple to coil outer surface using thermal paste
173. Connect coil to programmable DC power supply
174. Set current to rated maximum of 0.5A continuous
175. Monitor coil temperature rise from ambient baseline
176. Record temperature at 30-second intervals for 10 minutes
177. Verify steady-state temperature reached within 5 minutes
178. Check maximum temperature does not exceed 120°C limit
179. Calculate thermal resistance from power dissipation and temperature rise
180. Verify thermal resistance below 100°C/W maximum specification
181. Perform thermal cycling test: 10 cycles from -20°C to 85°C
182. Monitor resistance change throughout thermal cycling
183. Verify resistance returns to initial value after cycling
184. Check for any physical damage from thermal expansion/contraction
185. Inspect adhesive bonds remain intact after thermal stress testing

## Final Inspection and Packaging (Steps 186-195)

186. Perform final visual inspection under 10x magnification
187. Check for any wire damage, adhesive defects, or contamination
188. Verify lead wire attachment quality and positioning
189. Clean coil exterior using compressed air to remove any particles
190. Apply protective coating to lead wire solder joints if required
191. Place coil in ESD-protective packaging tray with foam cushioning
192. Attach QR code label with serial number and test data traceability
193. Seal package with moisture barrier bag including desiccant pack
194. Record final inspection results and approve for shipment to speaker assembly
195. Transfer completed voice coils to finished goods inventory with full documentation

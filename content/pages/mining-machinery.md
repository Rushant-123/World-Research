## Frame and Undercarriage Foundation

1. **Steel Plate Inspection**: Receive and inspect high-strength steel plates (50-100mm thick) for structural frame components, checking for mill certificates and material composition.

2. **Plate Cutting Preparation**: Clean steel plates and apply cutting layout patterns using CNC plasma cutting templates for main frame components.

3. **CNC Plasma Cutting**: Cut main frame rails, cross members, and structural plates using computer-controlled plasma cutting systems with ±2mm tolerance.

4. **Edge Preparation**: Grind and bevel cut edges to 30-45 degree angles for proper weld joint preparation on all structural components.

5. **Frame Rail Forming**: Use hydraulic press brakes to form main frame rails from 75mm thick steel plates, creating box-section structures.

6. **Fixture Setup**: Position frame components in precision welding fixtures that maintain alignment within 1mm across 8-meter lengths.

7. **Tack Welding**: Perform initial tack welds at 300mm intervals to hold frame components in proper alignment before full welding.

8. **Robotic Welding**: Execute continuous robotic GMAW welding on main frame seams using 1.6mm wire at 280 amps for full penetration.

9. **Manual Weld Finishing**: Complete complex joints and corners using manual SMAW welding with 5mm electrodes for difficult-to-reach areas.

10. **Stress Relief**: Heat treat welded frame assembly in 850°C furnace for 4 hours to relieve residual stresses in welds.

## Undercarriage Assembly

11. **Track Frame Fabrication**: Weld track frame assemblies from 50mm steel plates, creating the base structure for crawler systems.

12. **Roller Mounting**: Install lower rollers (8-10 per side) with sealed bearings rated for 200,000 hours of operation in contaminated environments.

13. **Idler Installation**: Mount front idler wheels with adjustment mechanisms allowing 150mm of track tension adjustment range.

14. **Sprocket Mounting**: Install drive sprockets with hardened teeth (58-62 HRC) connected to final drive assemblies through splined shafts.

15. **Track Link Assembly**: Connect individual track shoes (600-900mm wide) using hardened steel pins and sealed master links.

16. **Track Tensioning**: Adjust track tension using hydraulic cylinders to achieve proper sag of 50-75mm at mid-span between rollers.

17. **Final Drive Installation**: Mount planetary final drive units with 40:1 reduction ratios capable of 180,000 Nm output torque.

18. **Lubrication System**: Install automatic track lubrication system with reservoir and distribution lines to all pivot points.

19. **Undercarriage Testing**: Operate undercarriage on test track for 2 hours, checking for proper alignment, noise, and temperature rise.

20. **Frame to Undercarriage Mounting**: Attach main superstructure frame to undercarriage using eight high-strength bolts (M48 x 200mm) per side.

## Hydraulic System Installation

21. **Hydraulic Tank Mounting**: Install 800-liter hydraulic reservoir with baffles, filters, and level sensors on main frame.

22. **Main Pump Installation**: Mount variable displacement piston pumps (2-3 units) capable of 600 liters/minute at 350 bar pressure.

23. **Pump Drive Coupling**: Connect hydraulic pumps to engine PTO using flexible couplings rated for 500 Nm continuous torque.

24. **Control Valve Assembly**: Install main control valve bank with 8-12 independently controlled sections for various hydraulic functions.

25. **Pressure Relief Setting**: Adjust system relief valves to 380 bar maximum pressure with 350 bar normal operating limit.

26. **Hydraulic Line Routing**: Route high-pressure steel tubes (38mm diameter) from pumps to control valves with proper support clamps.

27. **Hose Assembly Installation**: Connect flexible hydraulic hoses rated for 420 bar working pressure to all moving components.

28. **Swing Motor Mounting**: Install hydraulic swing motor with 200:1 planetary reduction capable of 450 kNm output torque.

29. **Travel Motor Installation**: Mount hydraulic travel motors (one per track) with two-speed capability and automatic shifting.

30. **Boom Cylinder Mounting**: Install boom lift cylinders (bore 250mm, stroke 3000mm) with 850 kN lifting force capacity.

31. **Stick Cylinder Installation**: Mount stick cylinders (bore 220mm, stroke 2400mm) rated for 650 kN digging force.

32. **Bucket Cylinder Mounting**: Install bucket cylinders (bore 200mm, stroke 1800mm) providing 600 kN crowd force.

33. **Hydraulic Oil Filling**: Fill system with 1200 liters of ISO VG 46 anti-wear hydraulic oil meeting equipment manufacturer specifications.

34. **System Pressure Testing**: Pressurize complete hydraulic system to 420 bar for 30 minutes, checking for leaks at all connections.

35. **Flow Testing**: Verify hydraulic flow rates at each function using calibrated flow meters, ensuring proper pump output.

## Engine Installation

36. **Engine Bay Preparation**: Clean and prepare engine mounting area with precision-machined mounting pads for engine support.

37. **Diesel Engine Positioning**: Use overhead crane to position 6-cylinder turbocharged diesel engine (18-24 liters displacement) onto mounting frame.

38. **Engine Mounting**: Secure engine using vibration-isolating mounts rated for 8-ton engine weight and operating vibration.

39. **Cooling System Installation**: Mount radiator assembly (1.5m x 2m) with electric fan capable of 20,000 CFM airflow.

40. **Coolant Line Connection**: Connect reinforced rubber coolant hoses (75mm diameter) from engine to radiator with constant-tension clamps.

41. **Air Cleaner Mounting**: Install two-stage air filtration system with pre-cleaner and primary filter rated for dusty mining environments.

42. **Turbocharger Inspection**: Verify turbocharger installation and wastegate operation for proper boost pressure control (2.5-3.0 bar).

43. **Fuel Tank Installation**: Mount 1500-liter diesel fuel tank with baffles, pickup tubes, and level sensors.

44. **Fuel System Routing**: Install fuel supply and return lines with water separator and primary fuel filter assembly.

45. **Fuel Injection Testing**: Verify electronic fuel injection system operation and pressure (2000 bar common rail).

46. **Exhaust System Assembly**: Install exhaust manifold, turbocharger, catalytic converter, and muffler meeting emissions standards.

47. **Engine Wiring**: Connect engine control module (ECM) and all sensors including temperature, pressure, and speed sensors.

48. **Alternator Connection**: Wire 28-volt, 150-amp alternator to charging system with voltage regulation.

49. **Starting System**: Install dual 12-volt batteries (200 Ah capacity) in series for 24-volt starting system.

50. **Starter Motor Testing**: Verify electric starter motor operation (8 kW) with proper cranking speed.

## Boom and Stick Assembly

51. **Boom Section Fabrication**: Weld boom sections from 40mm steel plate using robotic welding for consistent quality.

52. **Stress Concentration Areas**: Reinforce high-stress areas around pin bosses with additional gusset plates and doubling plates.

53. **Boom Pin Bore Machining**: Precision machine pin bore holes to ±0.05mm tolerance for proper pin fit and rotation.

54. **Boom Pin Installation**: Install hardened steel pins (180mm diameter) with bronze bushings for articulation points.

55. **Boom Lubrication System**: Install automatic lubrication system with distribution block feeding all boom pivot points.

56. **Stick Fabrication**: Construct stick (dipper arm) from box-section steel with reinforced mounting points for bucket linkage.

57. **Stick Pin Boss Welding**: Weld heavy-duty pin bosses at stick attachment points using pre-heated steel (150°C preheat).

58. **Stick Linkage Assembly**: Install bucket linkage components including link arms and connecting pins with replaceable bushings.

59. **Quick Coupler Mounting**: Install hydraulic quick coupler system allowing tool changes without leaving operator cab.

60. **Bucket Attachment**: Mount mining bucket (6-12 cubic meter capacity) with replaceable wear plates and GET (ground engaging tools).

61. **Bucket Teeth Installation**: Install bolt-on bucket teeth with carbide tips rated for rock excavation and abrasive materials.

62. **Wear Plate Installation**: Bolt wear-resistant steel plates (400 HB hardness) to bucket floor and sides for extended life.

63. **Boom Geometry Verification**: Check boom and stick geometry to ensure proper reach, digging depth, and dump height specifications.

64. **Cylinder Stroke Testing**: Verify full stroke of all boom, stick, and bucket cylinders without interference or binding.

65. **Load Testing**: Apply static loads to boom structure to verify structural integrity (150% of rated lifting capacity).

## Swing System and Upperstructure

66. **Swing Bearing Installation**: Mount large-diameter slewing bearing (3-4 meter diameter) between undercarriage and upperstructure.

67. **Swing Bearing Torque**: Torque swing bearing mounting bolts (80-100 bolts) to specified value (1200 Nm) in proper sequence.

68. **Swing Ring Gear**: Install internal ring gear on swing bearing for swing motor pinion engagement (module 12-16 gear teeth).

69. **Swing Motor Mounting**: Position swing motor with pinion gear meshing properly with ring gear (0.2mm backlash).

70. **Swing Brake Installation**: Install spring-applied, hydraulically released brake capable of holding full machine on 30-degree slope.

71. **Upperstructure Frame Assembly**: Construct revolving frame from welded box-section steel forming platform for all upper components.

72. **Counterweight Mounting**: Install counterweight assembly (15-25 tons) at rear of upperstructure for balance during operation.

73. **Counterweight Fastening**: Secure counterweight using high-strength bolts with thread-locking compound and safety wiring.

74. **Swing Bearing Lubrication**: Install automatic lubrication system feeding grease to swing bearing raceways through distribution manifold.

75. **Swing System Testing**: Operate swing system through full 360-degree rotation, checking for smooth operation and proper speed.
